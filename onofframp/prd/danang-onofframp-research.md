# 调研笔记｜越南岘港（Da Nang）On/Off Ramp 市场与合规路径

> **调研主题**：越南岘港 VIFC 沙盒下的法币↔数币（On/Off Ramp）通道现状、主要机构与合规落地路径
> **调研日期**：2026-06-02 ｜ **最近更新**：2026-06-03
> **调研目的**：评估 BB/IPL/EX 在越南落地 OnRamp/OffRamp 的合规窗口与潜在合作方
> **结论可信度**：中高（基于公开法律/媒体/咨询机构报道，未含一手访谈）

---

## 0. 执行摘要（TL;DR）

- 越南 2025-06 立法承认数字资产，**2026-01-01 正式生效**；合法运营窗口当前仅限 **岘港、胡志明市两个 IFC 区**。
- **持牌交易所门槛极高**（资本金 ≈ 3.79 亿美元、外资 ≤ 49%、全国最多 5 家）→ 不是我们的现实路径。
- **沙盒试点是务实切入口**：参与方可获**责任豁免**、快速审批、税收激励。
- 岘港 On/Off Ramp 已有**两家代表机构**：**Basal Pay**（首张沙盒牌照、旅游 OffRamp）与 **MIMO**（非托管 USDT↔VND 双向）。
- **建议路径**：与本地已持牌沙盒主体（AlphaTrue / Dragon Lab）合作，而非自建越南持牌主体。

### 2.3 准入门槛对比

| 路径 | 资本金 | 外资上限 | 数量限制 | 适配我们? |

## 3. 主要 On/Off Ramp 机构

### 3.1 Basal Pay（开发方：AlphaTrue Solutions JSC）

> 备注：用户口中的 "alex pay" ≈ Basal Pay / AlphaTrue（"Alpha" 音近 "Alex"）。

| 维度 | 内容 |
|------|------|
| 里程碑 | **2025-08 获岘港首张沙盒牌照** — 越南首个境内"加密↔法币"兑换试点 |
| 业务 | 游客数秒内把 **USDT** 兑换为越南盾（VND），去中间环节 |
| 成本 | 较传统通道**降本约 30%** |
| 用户 | **5,000+ 注册用户**，约 **30% 国际游客** |
| 场景 | 旅游/消费支付（OffRamp 为主：USDT → VND 到店收款） |
| 规划 | 扩展至胡志明市 IFC，对接本地持牌交易所 |
| 创始人 | Dinh（AlphaTrue 创始人兼 CEO） |

**支付流程（PwC）**：游客出示付款 → 输入 VND 金额 → App 按实时汇率折算等值 USDT → 商户收到 VND 入账。

### 3.2 MIMO Solution（开发方：Dragon Lab JSC）

| 维度 | 内容 |
|------|------|
| 业务 | **非托管（non-custodial）**中介平台，**USDT ↔ VND 双向**兑换（On + Off Ramp） |
| 定位 | 纯法币↔数币通道，资金非托管 → 合规/安全叙事更强 |
| 状态 | 已进入岘港 VIFC 沙盒批准名单 |
| 意义 | 证明岘港接受"非托管 On/Off Ramp"模式在监管下运行 |

### 3.3 对比

| 维度 | Basal Pay | MIMO |
|------|-----------|------|
| 方向 | 偏 OffRamp（USDT→VND） | 双向 On/Off Ramp |
| 托管模式 | 平台中介 | **非托管** |
| 牌照状态 | 首张沙盒牌照（2025-08） | 已批准名单 |
| 验证场景 | 旅游消费（5000+ 用户） | 通道型 |
| 对我们参考 | 场景/获客验证 | 合规架构参考 |

## 5. 对 BB/IPL/EX 的启示

1. **沙盒优先于持牌**：资本金 + 外资上限主要卡持牌交易所；On/Off Ramp 走沙盒门槛低得多。
2. **非托管模式受认可**：MIMO 已验证，与我们 OnRamp/OffRamp 合规叙事一致。
3. **合作优于自建**：与 AlphaTrue / Dragon Lab 等已持沙盒牌照主体合作，落地更快。
4. **责任豁免红利**：沙盒内合规测试可豁免行政/民事责任，适合受控试点。
5. **场景已验证**：旅游/消费 OffRamp（USDT→VND 到店收款）有真实游客需求。
