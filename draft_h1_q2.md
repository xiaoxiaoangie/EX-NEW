  - 越南法规更新：FATF 灰名单未除名（2026/2 最新确认）；央行 Circular 09/2023 收紧非居民账户；新《数字技术产业法》2026/1 生效；AML Law 2025/11 生效

#### Remex 中越生鲜供应链方案（6/24 出炉）

- **场景**：中国采购商进口越南榴莲/海鲜
- **方案**：供应链公司垫资放款 → Remex 越南本地付款（VND）→ 中国采购商 RMB 出境或数币代付 → 供应链代扣还款
- **三种模式**：
  - **大额（账户模式）**：法币流程图（IPL 法币持牌方收汇/结汇/入账；EX 镜像账本）
  - **大额（数币模式）**：BB 数币资金中心归集
  - **小额（VCC 卡模式）**：放款/供应链公司提前充值、发卡给买家；主子账户体系；循环额度
- **EX 角色**：账本/主子账户、订单与发货数据中台、放款/还款编排、给 Remex 提供 API
- **资金角色约定**：EX 不碰钱，资金始终落在 IPL（法币）/BB（数币）/Remex（越南本地）/供应链公司（放款）
