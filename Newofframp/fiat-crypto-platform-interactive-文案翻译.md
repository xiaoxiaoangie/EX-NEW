## 14. 本地化规则

1. English 中统一使用 **Fiat Account** 表示法币账户，使用 **Crypto Wallet** 表示数币钱包。
2. 简体中文“充值”按术语表指商户从自有银行账户转入法币；繁体中文统一使用“同名充值”；English 使用 **Deposit**。
3. “充币 / 提币”分别使用 **Crypto Deposit / Crypto Withdrawal**，不得简写成 Deposit / Withdrawal。
4. “付款”统一使用 **Payout**；“收款”统一使用 **Collection**。
5. “大账户”统一翻译为 **Reference Account**；“虚拟账户”首次出现可写作 **Virtual Account (VA)**。
6. “SWIFT Code”在界面字段中统一写作 **BIC (SWIFT Code)**。
7. 交易唯一标识统一使用 **Transaction ID**；不得使用 MID、Order ID 等旧术语替代 `merchantNo` 或 `transactionId`。
8. 币种代码、银行名称、企业名称、账号、链上地址、交易编号和 Purpose Code 不翻译。
9. `SHA / OUR / BEN / KYC / KYB / KYT / AML / POBO / 2FA` 等行业缩写保持不变，首次出现时可补充英文全称。
10. 动态变量使用 `{变量}`，例如 `{金额} {币种}`、`返回{页面}页面`，三种语言中的变量名应保持语义一致。
