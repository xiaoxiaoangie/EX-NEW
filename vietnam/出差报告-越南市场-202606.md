# 出差报告 · 越南市场调研（支付 + Web3 + 跨境）

## 〇、TL;DR（核心结论）

1. **本地支付极强、外国人支付极弱。** 本地 NAPAS / VietQR 账户转账几乎零成本、秒到；但外国游客/外国主体支付体验很差——Alipay 旧模式用不了、外卡（含银联）受限、我们的 VCC 基本刷不动。**这是结构性缺口，也是机会。**
2. **Web3 是"高采用 + 强监管 + 刚起步合规"三重叠加。** 越南加密采用率常年全球第一梯队；但官方渠道极少，绝大多数走 **OTC / P2P + 黑市汇率**；政府 2025 起密集立法 + 严打非法 OTC（冻卡、关店）。
3. **合规落地窗口只在岘港（主要）/胡志明（消息）两个 IFC 沙盒。** 持牌交易所门槛极高（≈3.79 亿美元资本金、外资≤49%、5 年最多 5 家）；**On/Off Ramp 走岘港 VIFC 沙盒**（Basal Pay、MIMO、9Pay 等）是现实路径。
4. **真实需求三条线：出口收款（外国 VA）/ 侨汇与资金出境。** 都被现有合规渠道压抑，外溢到 OTC 与加密。
5. **我方现状：尚未约到 PSP/小银行类租户，手上都是直客、业务鱼龙混杂。** 需要尽快锁定合规载体（岘港沙盒合作方）+ 收敛客户画像。

## 一、越南本地支付方式现状

### 1.1 本地转账：NAPAS / VietQR —— 便宜、方便、统治级

- **NAPAS**（National Payment Corporation of Vietnam）是越南国家级跨行清算与转接网络；**VietQR** 是其统一二维码标准。【公开资料】
- 银行账户间 A2A 转账**近乎零费、实时到账**，扫 VietQR 即可，覆盖几乎所有银行与商户，**已成为国民级习惯**（远比卡支付普及）。
- 本地电子钱包 **MoMo、ZaloPay、VNPay** 在长尾零售场景补位，但底层仍高度依赖银行账户 + NAPAS。
- **含义**：越南是"账户/QR 直转"主导的市场，**不是卡组织主导**——这决定了外卡/VCC 天然水土不服。

### 1.2 外国人/外国主体支付：非常不愉快

| 痛点                    | 现状                                                                           | 说明                                                     |
| ----------------------- | ------------------------------------------------------------------------------ | -------------------------------------------------------- |
| **Alipay 旧模式** | 大量本地中小商户**收不了**外国 Alipay                                    | 历史上外国钱包与本地受理网络不互通【实地/道听途说】      |
| **外卡受理**      | Visa/Master 在中高端场景可用，**长尾商户少**；**银联卡很多不支持** | 越南受理网络以本地 QR 为主，外卡渗透低【实地/道听途说】  |
| **我们的 VCC**    | 在越南本地**基本刷不动**                                                 | 受理端不认/风控拒付，VCC 价值在越南受限【实地/道听途说】 |
| **游客换汇/支付** | 体验割裂，催生 OTC/加密换 VND 的灰色需求                                       | 见第二节 On/Off Ramp                                     |

### 1.3 正在改善：Alipay+ × NAPAS × 本地钱包互通

- **2025-04-03，NAPAS × Ant International（蚂蚁）× Vietcombank** 正式扩展**跨境 QR 互通**：中国游客可用 **Alipay 扫 VIETQRGlobal** 在 NAPAS 网络商户付款；越南人赴华也可用本地钱包扫码。【公开资料】
- 方向上 NAPAS 在把 VietQR 与 **中国（Alipay/微信系）、韩国、东盟（泰国等）** 逐步双向打通，**ZaloPay/MoMo 等本地钱包**也在接入跨境受理。【公开资料】
- **判断**：用户"Alipay 在推进跟 NAPAS、ZaloPay 合作"的说法**方向正确**——本质是**Alipay+ 跨境互通 + NAPAS 受理网络**，而非 Alipay 单独落地。具体到哪些钱包、哪些走廊已上线，**建议以 NAPAS/Ant 官宣为准（部分细节待核实）**。

> **机会点**：外国人/外国主体在越南的"最后一公里支付"仍是缺口；谁能把**境外资金/稳定币 → 合规 VND 受理（VietQR）**打通，谁就吃到游客 + 跨境消费红利（这正是岘港沙盒 On/Off Ramp 在做的事）。

### 1.4 本地加密钱包 × NAPAS：链上余额直连本地收付

- **越南本地有大量加密钱包**，且不少已**对接 NAPAS 做本地下发（payout）与收款（collection）**——用户的**链上/稳定币余额可直接经 NAPAS/VietQR 完成 VND 的收与付**，体感上"加密钱包 = 一个能扫码、能转账的本地支付账户"。【实地/道听途说 + 公开资料】
- **年轻人群高频使用加密钱包**：在年轻用户中，**加密钱包（含稳定币）已是日常持有/收付的常见工具**，与本地银行 App / 电子钱包并存使用。【实地/道听途说】
- **本地收付通道 + NAPAS 接入是钱包的"刚需底座"**：越南这些数币钱包**几乎都必须配本地 VND 收款/付款通道、且必须对接 NAPAS/VietQR** 才能用——否则钱包里的链上/稳定币余额无法变成"能扫码、能转账、能落地花出去"的本地购买力。换言之，**"本地收付 + NAPAS" 不是加分项，而是钱包能否被日常使用的准入门槛**。【实地/道听途说】
  - **对我方的指向**：这恰好是 EX 的能力切口——若我方能向本地钱包/钱包服务商**输出"VND 本地收付通道 + NAPAS 受理接入 + USDT 承兑/流动性"的合规组件**（经岘港沙盒持证主体兜底），即可成为这些钱包的**底层收付与 On/Off Ramp 引擎**，而非与其正面竞争 C 端。
- **机制含义**：这其实是**"软性 On/Off Ramp 嵌入支付"**——钱包侧持币、NAPAS 侧落 VND，**把 OTC 的换汇动作隐藏进了收付流程**。这既解释了 USDT 为何深度渗透日常，也说明**"加密钱包 + 本地收付网络"的组合在越南已被市场验证**。
- **对我方的指向**：我方的 **USDT 流动性 + 承兑能力**若能接上**本地钱包 / NAPAS 收付**（经岘港沙盒持证主体合规化），即可复制这一被验证的形态；同时需警惕——**未挂靠牌照的"钱包直连 NAPAS 收付"正是监管严打的灰色地带**（冻卡风险，见 2.3）。

> ⚠️ **待核实**：具体是哪些本地钱包、以何种方式接入 NAPAS（直连/经持牌 PI 中转）、是否已纳入沙盒——建议现场点名核实钱包清单与合规路径。

## 二、Web3 行情与政策

### 2.1 市场行情：高采用、黑市汇率、OTC 主导

- 越南**加密采用率常年全球第一梯队**，估计持币人群上千万级；交易所以 **Binance / Bybit / OKX 的本地化 P2P** 为主要入金/出金路径。【公开资料】
- **黑市汇率 + OTC 做交易**普遍：官方/银行汇率与自由市场汇率存在明显价差，**USDT 成为事实上的"美元替代 + 跨境结算媒介"**，OTC 报价常优于银行。

### 2.2 谁才能"走官方"？—— 目前几乎没有零售官方通道

- 现阶段**没有面向零售的合法加密兑换通道**（持牌交易所尚在牌照申请期，零售试点仅限岘港沙盒场景）。
- 想"走官方"的现实选项：
  1. **岘港 VIFC 沙盒**内的持牌/获批主体（Basal Pay、MIMO、9Pay 等）——但多为**特定场景**（游客消费 OffRamp、稳定币 QR 支付）。
  2. 未来**5 家持牌交易所**落地后（门槛见 2.5）。
- 

### 2.3 监管打击：冻卡、关店、账户清理

- **针对非法 OTC**：监管 + 银行联合**冻结涉加密资金账户、查处/关停线下 OTC 点**，力度持续加大。**关停具体家数待核实**。【实地/道听途说】
- **银行账户大清理（需与 OTC 打击区分）**：2025 年生物识别实名新规下，越南**大规模关闭/冻结沉睡及不合规账户（公开报道达数千万级）**，主要目的是反诈反洗钱，但**客观上加大了 OTC 用户"动卡冻钱"的风险**。【公开资料】
- **趋势**：合规框架落地的同时，**对灰色 OTC 的执法只会更严**——"开正门、堵偏门"同步进行。

### 2.4 强烈的结汇需求

- 出口企业、跨境电商卖家、外贸收款方有大量**外币→VND 结汇 / VND→外币购汇**需求；正规结汇受额度与单据约束，**部分需求外溢到 USDT/OTC**。【实地/道听途说 + 公开资料】
- **这是 On/Off Ramp + 合规结汇通道的核心商业场景之一**。

### 2.6 岘港（Da Nang）VIFC 沙盒：现实落地窗口 + 机构/牌照说明

- **为什么是岘港**：岘港是越南**首个**获准对加密相关方案做**受控测试**的城市；VIFC 沙盒内合规测试**可豁免行政/纪律/民事责任**（须遵守批准流程），并享更快审批与税收/投资激励。【公开资料】
- **MIMO 是什么**：**MIMO Solution（开发方 Dragon Lab JSC）**——**非托管（non-custodial）**的 **USDT ↔ VND 双向兑换（On + Off Ramp）**中介平台，已进入**岘港 VIFC 沙盒批准名单**。其意义是：**证明"非托管 On/Off Ramp"模式可在越南监管下合法运行**。【公开资料】
- **哪些机构可以合法做**：当前合法做加密↔法币的，主要是**进入岘港 IFC 沙盒首批名单的获批主体**（如 Basal Pay、MIMO、9Pay、PayD 等）；零售层面**只在沙盒场景内合法**，全国性零售合规通道仍待持牌交易所落地。

## 三、其他需求与趋势

### 3.1 本地电商 / 跨境电商高速发展

- Shopee、TikTok Shop、Lazada 等驱动**本地电商爆发**；**跨境电商**（卖家出海 + 海外买家入越）增长迅猛。【公开资料】
- 衍生支付需求：跨境收款、批量分账、卖家结算、退款/拒付处理。

### 3.2 出口贸易扩张 → 外国 VA 需求上升

- 越南是**制造/出口大国**，出口贸易与跨境贸易**持续扩张**。
- **趋势**：对**其他国家 VA（虚拟账户/收款户）**的需求会增加（出口商要在欧美/东亚收款）。
- **现实**：**本地支付机构对接其他国家 PSP 并不难**——即"越南侧落地不是瓶颈，瓶颈在海外收单/收款侧的合规与账户"。【实地/道听途说】
- **含义**：我方可主打"**为越南出口商提供多国 VA + 收款 → 结汇/出金到 VND**"的组合能力。

### 3.3 侨汇与资金出境需求

- 越南是**全球前列的侨汇接收国**（年入境侨汇规模庞大），同时**大量个人/企业有"资金出境"诉求**（"很多人钱想出去"）。【公开资料 + 实地/道听途说】
- 外汇管制下出境受限 → **再次外溢到 USDT/OTC**；这是 OTC 屡禁不止的根本动因之一。

## 四、服务机构与客户画像观察（我方现状）

1. **尚未约到"PSP / 小银行"这类客户画像。** 目前缺少能成为**合规载体/渠道**的机构型租户。
2. **手上都是直客，业务鱼龙混杂。** 直客需求分散、合规质量参差，**难以标准化产品 + 难以集中合规**。
3. **风险**：直客 + 灰色 OTC 需求混杂，若无合规载体兜底，**KYC/AML 与冻卡风险会传导给我方**。

## 五、对 EX / BB / IPL 的启示与建议

1. **合规载体优先：尽快绑定岘港沙盒持证主体**（MIMO / Basal Pay / 9Pay 之一），用其牌照做**USDT↔VND 本地合规落地**，我方提供 USDT 流动性 + 海外侧能力。对应承兑业务"垫 U（模式 B）"更优解。
2. **产品打"外国人/外国主体支付缺口"**：境外资金/稳定币 → 合规 VND（VietQR 受理）；以及出口商**多国 VA 收款 → 结汇 VND**。
3. **客户画像收敛**：从"鱼龙混杂的直客"转向**出口电商/跨境卖家/侨汇相关**等可标准化、可合规的细分；同时主动开发 **PSP/小银行**类机构租户做合规载体。
4. **强合规叙事**：越南执法趋严（冻卡/关 OTC），**"合规 + 不动卡 + 可追溯"本身就是卖点**——正对应我方"合规运营平台"定位。
5. **避免触碰零售非法 OTC**：所有 On/Off Ramp 必须挂靠沙盒持证主体，**不做无牌零售兑换**。
