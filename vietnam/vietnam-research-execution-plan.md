# 越南市场调研 & 竞品调研落地方案（作战手册）

> 角色主导：商业化产品经理（统筹方法论 + 模板 + 归档）；全员参与（产品 / 销售 / 运营 / 合规）
> 目的：把市场调研、竞品调研从"脑子里 / 微信里"**全部落到纸面、统一成文档**，沉淀为可复用的知识资产
> 对齐：`H2-plan-2026.md`（Q3 7-8 月调研里程碑）、`vietnam-overall-solution.md`、`ex-three-layer-solution.md`、`vietnam-payment-web3-report.md`
> 周期：Q3（7-8 月为主，持续更新）

---

## 〇、一页纸（TL;DR）

1. **三类调研**：**市场调研**（看清环境与机会）+ **竞品调研**（看清对手怎么做）+ **客户调研**（看清谁能合作）。
2. **三种方法**：**桌面研究**（公开资料）+ **友商卧底**（伪装客户走全流程）+ **客户田野**（实地拜访 30+）。
3. **一套模板，人人落纸**：竞品画像卡 / 客户画像卡 / 市场快照 / 通道测评卡——**谁调研谁填卡**，统一归档，不口头汇报。
4. **三份成果**：① 越南市场调研报告；② 竞品（友商）通道与痛点白皮书；③ 客户画像库 + 解决方案映射。
5. **分工不丢项**：产品（方法论+模板+产品类调研+汇总成文）、销售（客户/竞品商务情报）、运营（通道/成本/流水）、合规（监管/牌照/风险）。

---

## 一、调研目标与范围

### 1.1 三大调研目标

| 调研 | 回答的问题 | 服务于 |
| --- | --- | --- |
| **市场调研** | 越南市场环境、监管走向、客户盘子、机会与红线在哪 | 战略定位、客户画像 |
| **竞品调研** | 本地派 / 外地派 / 中港同源机构**做什么、怎么做、交易量多少、死穴在哪** | 差异化、定价、风控壁垒 |
| **客户调研** | 哪类客户能合作、痛点是什么、能否转租户 | 销售展业、解决方案设计 |

### 1.2 范围边界

- **聚焦**：越南跨境支付 / 承兑（VND↔USDT、跨境收付）相关主体与场景。
- **产品视角主线**（我负责）：把调研结论转化为**产品形态、客户画像、解决方案**。
- **其他人调研**（销售/运营/合规）：必须按本手册模板**落卡归档**，由产品侧汇总成文。

---

## 二、调研对象分类

### 2.1 竞品（友商）三派

| 派系 | 定义 | 重点看 |
| --- | --- | --- |
| **本地派** | 越南本土 OTC / PSP / 钱包 | 本地通道、银行关系、合规牌照、冻卡应对 |
| **外地派** | 非中港的外资机构 | 打法、客群、产品形态 |
| **中港同源派** | 与我们一样来自中国 / 香港的机构 | **直接对手**：通道成本、报价、客群、交易量、合规外壳 |

> 已知样本（持续补充）：Holdstation / 9Pay(=南配) / Dragon Lab(MIMO) / Nanpei / Alixpay 等——详见 `vietnam-payment-web3-report.md`。

### 2.2 客户三类（对齐 H2 客户画像）

| 客户 | 现状 | 调研重点 |
| --- | --- | --- |
| **OTC（潜在租户）** | 接触最多 | 量、资金来源、合规意愿、痛点、能否提纯转租户 |
| **正规跨境企业** | 种子机会 | 跨境场景、合规凭证需求、回款链路 |
| **本地 PSP / 银行** | 计划目标 | 合作可能、能力缺口、主体隔离需求 |

### 2.3 市场环境维度

监管与牌照走向、本地银行与清算（NAPAS/VietQR）、冻卡与执法、汇率与 USDT 结汇、岘港沙盒等政策窗口。

## 三、调研方法

| 方法 | 做法 | 负责 | 产出 |
| --- | --- | --- | --- |
| **桌面研究** | 公开资料、官网、新闻、监管文件、链上数据 | 产品 + 合规 | 市场快照、竞品基础画像 |
| **友商卧底** | 伪装客户走通 3-5 家友商**全流程**（开户→KYC→入金→结汇→出金），摸清防冻卡、通道、成本、死穴 | 产品 + 运营 + 前线 | 通道测评卡、白皮书 |
| **客户田野** | 实地拜访 30+ 承兑大客 + 本地二三线 PSP，咨询式访谈 | 销售 + 产品 | 客户画像卡、痛点清单 |

> **卧底纪律**：统一话术、统一记录字段、留存截图/费率/时效证据；**完成即填通道测评卡**，不靠回忆。

## 四、统一输出模板（人人落纸的核心）

> **原则：谁调研谁填卡，卡片即文档**。所有卡片归档到 `market-research/` 下统一目录，命名规范见 §六。

### 4.1 竞品画像卡（每家友商一张）

```
# 竞品画像｜<机构名>
- 派系：本地 / 外地 / 中港同源
- 主体与牌照：<公司主体 / 持牌情况 / 合规外壳>
- 业务范围：<OTC / PSP / 钱包 / 承兑 / 跨境收付>
- 目标客群：<谁在用>
- 通道与币种：<收/付币种、银行、链>
- 报价 / 费率：<markup / setup / 阶梯>
- 交易量（估）：<日/月量级 + 来源>
- 防冻卡 / KYC 做法：<手段>
- 死穴 / 短板：<痛点>
- 对我们的启示：<差异化点>
- 信息来源：【公开 / 一手 / 卧底】 + 日期 + 调研人
```

### 4.2 客户画像卡（每个目标客户一张）

```
# 客户画像｜<客户名>
- 类型：OTC / 正规跨境企业 / PSP-银行
- 业务与场景：<做什么生意、跨境流向>
- 量级：<月交易量 + 币种>
- 资金来源 / 合规度：<可核实程度>
- 痛点：<冻卡 / 凭证 / 成本 / 时效>
- 合规意愿：<高/中/低>
- 能否转租户：<判断 + 准入档（全合规/半合规）>
- 对应方案：<匹配的解决方案>
- 跟进状态：<接触 / 调研 / 商务 / 签约>
- 调研人 + 日期
```

### 4.3 通道测评卡（卧底/实测一条通道一张）

```
# 通道测评｜<机构-通道>
- 全流程步骤 + 各步耗时
- 开户/KYC 门槛与资料
- 入金/出金方式（VietQR / 银行 / 链）
- 实测费率 + 隐性成本
- 防冻卡机制
- 到账时效 + 失败率（观察）
- 证据：截图/记录
- 测评人 + 日期
```

### 4.4 市场快照卡（按维度）

```
# 市场快照｜<维度，如：监管走向 / NAPAS / 冻卡执法>
- 现状事实 + 数据
- 趋势判断
- 对我们的影响（机会 / 红线）
- 来源 + 日期
```

## 五、分工与责任（不丢项）

| 角色 | 负责调研 | 落纸产出 |
| --- | --- | --- |
| **产品（我）** | 方法论 + 模板；产品类调研（形态/画像/方案）；**汇总成文** | 本手册 + 三份成果文档 + 模板维护 |
| **销售** | 客户商务情报、竞品报价/客群 | 客户画像卡、竞品画像卡（商务字段） |
| **运营** | 通道成本、流水、时效、冻卡实测 | 通道测评卡 |
| **合规** | 监管/牌照/风险/资金来源 | 市场快照卡（监管类）、风险清单 |

> **关键机制**：其他人调研**必须按模板交卡**；产品侧每周汇总，缺卡即催。**口头/微信信息不计入，未落纸 = 未完成。**
