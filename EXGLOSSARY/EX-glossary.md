## 2. 订单与交易术语

### 2.1 订单层级

| 简体中文   | 繁體中文（粵語） | English                 | 说明                                                                                            |
| ---------- | ---------------- | ----------------------- | ----------------------------------------------------------------------------------------------- |
| 商户业务单 | 商戶業務單       | Merchant Biz Order      | 商户自有系统的订单，通过 API 传入 EX 平台，携带商户侧的业务订单号（bizOrderNo）。代表商户的业务意图 |
| 业务订单   | 業務訂單         | Biz Order               | 调单（RFI）场景下客户需要录入的自身业务订单，如外贸订单、物流订单等                                |
| 订单       | 訂單             | Order                   | EX 平台系统生成的订单，对应商户的一次业务请求。商户端可见的核心单据，标识为 transactionId           |
| 交易单     | 交易單           | Transaction             | EX 系统内部的账务记录，按 SP 拆分，包含计费、汇率、风控、记账；对外统一使用 transactionId 作为唯一标识 |
| 渠道单     | 渠道單           | Channel Request         | SP 向底层渠道发起的执行请求记录，仅 SP 后台可见                                                 |
| 三单模型   | 三單模型         | Three-Layer Model       | Order → Transaction → Channel Request 的三层单据架构                                          |
| 交易单链   | 交易單鏈         | Transaction Chain       | 一个 Order 下按顺序执行的多笔 Transaction 序列                                                  |

### 2.2 订单标识

> **核心三要素**：`merchantNo`（商户维度） + `bizOrderNo`（商户系统订单维度） + `transactionId`（EX 平台交易维度）。
> 以上字段名为 API / 回调 / 对账中的**统一规范**，禁止使用 MID、Order ID、orderNo 等旧命名。

| 简体中文       | 繁體中文（粵語） | English        | 字段名           | 说明                                                                                  |
| -------------- | ---------------- | -------------- | ---------------- | ------------------------------------------------------------------------------------- |
| 商户号         | 商戶號           | Merchant No    | `merchantNo`     | EX 平台为商户分配的唯一标识，替代旧的 `MID` / 客户 ID                                 |
| 商户业务订单号 | 商戶業務訂單號   | Biz Order No   | `bizOrderNo`     | 商户自有系统的业务订单号，由商户通过 API 传入，同一商户下唯一，用于幂等和对账          |
| 交易 ID        | 交易 ID          | Transaction ID | `transactionId`  | EX 平台生成的交易唯一标识，商户端可见，用于查询、对账、回调；替代旧的 `Order ID`       |
| 渠道单号       | 渠道單號         | Channel Ref ID | —                | SP 或渠道侧返回的交易参考标识，仅内部对账使用（字段名暂不统一）                         |

### 2.3 订单类型

| 简体中文 | 繁體中文（粵語） | English           | 说明                                                                         |
| -------- | ---------------- | ----------------- | ---------------------------------------------------------------------------- |
| 预约单   | 預約單           | Reservation Order | 商户先下单预约，等银行转账到账后自动触发承兑的订单                           |
| 即时单   | 即時單           | Instant Order     | 使用法币钱包余额直接扣款，实时完成承兑的订单                                 |
| 消费订单 | 消費訂單         | Spending Order    | VCC 卡片在商户 POS/线上消费产生的交易订单（授权→清算→结算）                |
| 扣费订单 | 扣費訂單         | Fee Order         | 平台向商户收取费用的订单。场景包括：VA 收费、卡片手续费、强制扣费等          |
| 资金订单 | 資金訂單         | Fund Order        | VCC 卡账户的资金操作订单，包括：卡金额转入（Top-Up）、卡金额转出（Withdraw） |
| 挂账     | 掛賬             | Suspense          | 收到款项但无法匹配到预约单时，资金暂挂在账户中                               |

### 2.4 扣费订单场景

| 简体中文   | 繁體中文（粵語） | English               | 说明                                                    |
| ---------- | ---------------- | --------------------- | ------------------------------------------------------- |
| VA 收费    | VA 收費          | VA Fee Deduction      | 虚拟账户维护费、VA 开户费等平台向商户收取的 VA 相关费用 |
| 卡片收费   | 卡片收費         | Card Fee Deduction    | 开卡费、年费、卡管理费等平台向商户收取的卡片相关费用    |
| 强制扣费   | 強制扣費         | Forced Deduction      | 因合规、风控、争议处理等原因，平台从商户账户强制扣款    |
| 服务费扣费 | 服務費扣費       | Service Fee Deduction | 按约定的费率方案定期或逐笔收取的服务手续费              |
| 罚款扣费   | 罰款扣費         | Penalty Deduction     | 因违规操作、超限使用等原因产生的罚款扣费                |

## 3. 账户与资金术语

### 3.1 账户类型

| 简体中文   | 繁體中文（粵語） | English              | 说明                                                             |
| ---------- | ---------------- | -------------------- | ---------------------------------------------------------------- |
| 法币账户   | 法幣賬戶         | Fiat Account         | 持有法定货币余额的账户，按 SP 区分（BB 法币账户 / IPL 法币账户） |
| 数币钱包   | 數幣錢包         | Crypto Wallet        | 持有数字货币的钱包，支持充币/提币，由 BB 提供                    |
| 虚拟账户   | 虛擬賬戶         | Virtual Account (VA) | 银行分配的专属虚拟收款账号，用于同名收款，一户一号               |
| 大账户     | 大賬戶           | Reference Account    | VA 背后的主账户，多个 VA 归属同一大账户，资金最终汇集于此       |
| 代收付账户 | 代收付賬戶       | Omnibus Account      | BB 自有的统一收付款账户，多个商户共用同一账号                    |
| 中间户     | 中間戶           | Transit Account      | BB 和 IPL 之间用于跨 SP 资金划转的清算中间账户                   |
| 冻结       | 凍結             | Hold                 | 交易发起时锁定账户中对应金额，防止重复使用                       |
| 余额       | 餘額             | Balance              | 账户中可用的资金数量                                             |
| 资金调拨   | 資金調撥         | Internal Transfer    | IPL 与 BB 之间定期进行的资金轧差和划转                           |

#### 3.1.1 术语使用规范：Currency / Assets / Wallets / Accounts

> **原则：法币用 Account，数币用 Wallet；币种用 Currency，持有物用 Assets。**

| 场景 | 正确用法（zh） | 正确用法（hk） | 正确用法（en） | 说明 |
| ---- | -------------- | -------------- | -------------- | ---- |
| 选择具体货币单位 | 币种 | 幣種 | Currency | 下拉选择器、表头筛选，如 USD、USDT |
| 用户持有的资金总称 | 资产 | 資產 | Assets | 菜单名称、概览页标题，含法币+数币+卡 |
| 加密类资产统称 | 加密资产 | 加密資產 | Crypto Assets | 仪表盘资产分类标签 |
| 法币余额容器 | 法币账户 | 法幣帳戶 | Fiat Account(s) | 持有法币余额的账户 |
| 数币余额容器 | 数币钱包 | 數幣錢包 | Crypto Wallet(s) | 持有数币余额的钱包 |
| 资产总览页（法币+数币） | 资产列表 | 資產列表 | Assets Overview | 展示所有账户和钱包的列表页 |

**禁止混用：**
- ❌ "加密资产账户" → ✅ "数币钱包"（数币用 Wallet，不用 Account）
- ❌ "法币钱包" → ✅ "法币账户"（法币用 Account，不用 Wallet）
- ❌ 菜单和页面标题用 "Wallets" 表示资产总览 → ✅ 用 "Assets" 或 "资产列表"

### 3.2 资金操作（充值 / 提现 / 充币 / 提币 / 买卖数币）

| 简体中文 | 繁體中文（粵語） | English           | 说明                                                                 |
| -------- | ---------------- | ----------------- | -------------------------------------------------------------------- |
| 充值     | 同名充值         | Deposit           | 商户从自有银行账户向平台法币账户转入资金（汇款人 = 商户）            |
| 提现     | 提現             | Withdrawal        | 从平台法币账户提取资金到商户银行账户                                 |
| 充币     | 充幣             | Crypto Deposit    | 从外部链上地址向平台数币钱包转入数字货币                             |
| 提币     | 提幣             | Crypto Withdrawal | 从平台数币钱包向外部链上地址转出数字货币                             |
| 买入数币 | 買入數幣         | Buy Crypto        | 使用法币余额购买数字货币（如 USD → USDT），法币账户扣减，数币入钱包 |
| 卖出数币 | 賣出數幣         | Sell Crypto       | 卖出数字货币获得法币（如 USDT → USD），数币从钱包扣减，法币入账户   |

### 5.2 支付通知字段（Payment Notification Payload）

> 支付回调 `notifyData` 的标准字段定义，适用于所有支付/退款/代付通知场景。

#### 5.2.1 主体与订单标识

| 字段名          | 类型   | 简体中文       | 说明                                                       |
| --------------- | ------ | -------------- | ---------------------------------------------------------- |
| `productCode`   | string | 产品代码       | 产品线代码（如 `CHECKOUT` / `POBO` / `COBO` / `VCC`）       |
| `merchantNo`    | string | 商户号         | EX 为商户分配的唯一标识（§2.2）                             |
| `bizOrderNo`    | string | 商户业务订单号 | 商户系统订单号，商户侧唯一（§2.2）                          |
| `transactionId` | string | 交易 ID        | EX 平台生成的交易唯一标识（§2.2）                           |
| `transactionType` | string | 交易类型     | 见枚举【交易类型】：Payment / Refund / Payout / Deposit 等 |
| `notifyType`    | string | 通知类型       | 见枚举【交易类型】，与 `transactionType` 对应               |

#### 5.2.2 金额与币种

| 字段名            | 类型   | 简体中文     | 说明                                                       |
| ----------------- | ------ | ------------ | ---------------------------------------------------------- |
| `payerCurrency`   | string | 付款币种     | 付款方使用的币种                                            |
| `payerAmount`     | number | 付款金额     | 付款方实际支付金额                                          |
| `payeeCurrency`   | string | 收款币种     | 收款方到账的币种                                            |
| `payeeAmount`     | number | 实际到账金额 | 收款方最终到账金额                                          |
| `rate`            | number | 汇率         | 付款币种 ⇌ 收款币种的换汇汇率                               |
| `actualDeductAmount` | number | 实际扣款金额 | 钱包/账户实际被扣除的金额（含费用）                      |

#### 5.2.3 状态与时间

| 字段名         | 类型   | 简体中文 | 说明                                                    |
| -------------- | ------ | -------- | ------------------------------------------------------- |
| `status`       | string | 交易状态 | 见枚举【交易状态】                                      |
| `completeTime` | string | 完成时间 | 交易最终完成的时间戳                                    |
| `resultMessage`| string | 错误描述 | 失败/异常场景下的错误描述                                |
| `description`  | string | 补充说明 | 待补充材料等特殊状态下返回的说明信息                     |

#### 5.2.4 费用相关

| 字段名              | 类型         | 简体中文           | 说明                                                              |
| ------------------- | ------------ | ------------------ | ----------------------------------------------------------------- |
| `transactionFee`    | number       | 交易手续费         | 本笔交易实际收取的手续费                                           |
| `feeCurrency`       | string       | 手续费币种         | 手续费结算币种；手续费为 0 时此字段不传                             |
| `poboFee`           | number       | POBO 费用          | 代付（Payment On Behalf Of）产生的费用，仅 POBO 场景返回            |
| `poboFeeCurrency`   | string       | POBO 费用币种      | POBO 费用币种，POBO 费用非 0 时返回                                 |
| `totalFeeAmount`    | number       | 费用总金额         | 所有费用项合计，等于 `fees[].amount` 之和                           |
| `fees`              | array[object]| 费用明细列表       | 费用逐项明细（费用名称、类型、金额、币种等）                        |

#### 5.2.5 退款相关

| 字段名              | 类型   | 简体中文       | 说明                                    |
| ------------------- | ------ | -------------- | --------------------------------------- |
| `refundAmount`      | number | 退款金额       | 本次退款的金额                           |
| `refundCurrency`    | string | 退款币种       | 退款币种                                 |
| `refundFee`         | number | 退款手续费     | 退款操作收取的手续费                     |
| `refundFeeCurrency` | string | 退款手续费币种 | 退款手续费结算币种                       |

## 6. 费率与汇率术语

| 简体中文   | 繁體中文（粵語） | English           | 说明                                          |
| ---------- | ---------------- | ----------------- | --------------------------------------------- |
| 汇率       | 匯率             | FX Rate           | 两种货币之间的兑换比率                        |
| 预汇率     | 預匯率           | Indicative Rate   | 下单时展示的参考汇率，非最终结算汇率          |
| 结算汇率   | 結算匯率         | Settlement Rate   | 实际结算时使用的汇率                          |
| 锁定汇率   | 鎖定匯率         | Locked Rate       | 即时单下单时锁定的汇率，不随市场波动          |
| 实时汇率   | 實時匯率         | Live Rate         | 当前市场的即时汇率                            |
| 换汇       | 換匯             | FX                | 法币账户内不同币种之间的兑换（如 USD ↔ EUR） |
| 承兑费     | 承允費           | Conversion Fee    | 执行法币与数币承兑时收取的手续费              |
| 收款手续费 | 收款手續費       | Collection Fee    | VA 或代收付账户收款时收取的手续费             |
| 网络费     | 網絡費           | Gas Fee           | 链上转账产生的区块链网络费用                  |
| 固定费     | 固定費           | Flat Fee          | 不随交易金额变化的固定手续费                  |
| 比例费     | 比例費           | Percentage Fee    | 按交易金额百分比计算的手续费                  |
