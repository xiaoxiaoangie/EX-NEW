# 下半年（H2）商业产品业绩计划

> 角色：商业产品经理。本计划以 **"重点项目商业化落地 + 交易量"** 为主线，配套 **渠道健康支撑** 与 **质量优先的租户转化**，并明确风险底线与团队节奏。

---

## 一、计划思路：为什么这样定指标

在定目标之前，先明确 EX 的业务特性——它决定了指标的设计方式：

- **EX 是 To 租户 / SP 的平台模式**：我们通过租户、SP 间接触达终端用户，**不是直客拓展**。因此客户数量天然有限，**不能用"客户数量"作为核心业绩指标**。
- **现状**：意向租户 40+，但**大部分难以转化**。所以拓展的重点是**转化质量与单租户产能**，而不是堆数量。
- **结论**：H2 的业绩应围绕三件事——
  1. **重点项目商业化落地**：以**交易量**为最终结果指标；
  2. **渠道建设**：以**上线 + 健康运营 + 0 风险事件**为支撑指标；
  3. **租户转化质量**：以**"真正产生交易量的租户数"**为有效指标。

**指标设计三原则**（贯穿全文）：

- **先行指标**（过程，可控）：上线时间节点、渠道对接完成、租户签约；
- **滞后指标**（结果，看产出）：交易量；
- **质量指标**（底线，不可破）：0 渠道风险事件、0 资金风险事件。

## 三、分项展开

### 1. 重点项目商业化（核心：交易量）

| 项目                       | H2 目标               | Q3 交易量                       | 关键先行节点                           |
| -------------------------- | --------------------- | ------------------------------- | -------------------------------------- |

| **结汇 SP**          | 7 月底交付并跑量      | **1,000 万**（含 A 渠道） | SP 上线 → 渠道就绪 → 交易量爬坡      |

### 2. 渠道建设（支撑能力 + 风险底线）

| 渠道                                                        | 目标               | 风控底线                                 |
| ----------------------------------------------------------- | ------------------ | ---------------------------------------- |
| **越南渠道**（本地收付、OTC、全合规通道、半合规通道） | 对接上线并健康运营 | **0 渠道风险事件、0 资金风险事件** |
| **结汇渠道**                                          | 对接上线并健康运营 | **0 渠道风险事件、0 资金风险事件** |
| **美国 Kabin 通道**                                   | 对接上线并健康运营 | **0 渠道风险事件、0 资金风险事件** |

> 渠道是项目交易量的**前置条件**——渠道未健康上线，对应项目的交易量目标无法达成。建议把"渠道上线时间"作为项目交易量的**先行里程碑**绑定考核。

### 3. 租户 / 客户转化（质量优先，而非数量）

鉴于平台模式 + 40+ 意向多数难转化，建议：

- **对意向租户分层**：高 / 中 / 低转化可能，集中资源攻高可能；
- **核心指标改为"产生交易量的租户数"**，而非"签约数"或"意向数"——真正贡献交易量的租户才算有效；
- **建议设定**：H2 新增有效租户（已产生交易量）**≥ X 个**（请补具体数字）。

**法币租户：用价格换转化（H2 重点抓手）**

- H2 **法币业务的租户有望进来几个**，但**前提是价格需要适当下调**——当前价格是转化的主要阻力。
- 这是一次**主动的定价策略**：用**适度降价**换取**租户进件与交易量起量**，先把量做起来、再谈利润。
- 建议明确：
  - **降价幅度 / 目标价位**（建议给出区间，并测算对 take rate 的影响）；
  - **降价的边界**——只对法币租户、或只在 H2 起量期，避免无差别降价；
  - **目标**：H2 法币新增有效租户 **≥ N 个**（请补具体数字），并贡献相应交易量。

> **逻辑**：法币租户对价格敏感，**先用价格把量引进来**，符合"先有业务量、再谈扩张与利润"的整体节奏。

### 4. 新兴市场探索（探索性目标）

- 选定 **1–2 个**候选新兴市场，完成**可行性 / 商业化方案预研**（监管、渠道、需求、竞品）；
- 这是探索性投入，**设过程指标**（预研产出物 / 决策结论），**不背交易量**。

### 5. 团队建设（条件性）

- **触发条件**：业务起色后（例如某重点项目跑出交易量）再建设团队；
- **H2 计划**：条件性**落地 1 人**——以业务量为前提，避免在没有产出时增加固定成本。

## 四、Q3 / Q4 节奏建议

| 阶段         | 重心               | 主要目标                                                               |
| ------------ | ------------------ | ---------------------------------------------------------------------- |
| **Q3** | **上线为主** | 越南方案、结汇 SP 上线；越南 / 结汇 / Kabin 渠道对接上线；卡台客户签约 |

> 这样拆分能让"先行指标（上线）"在 Q3 兑现、"滞后指标（交易量）"在 Q4 释放，避免把所有交易量压力堆在年底。

## 五、让计划"更好"的几点建议

1. **交易量币种已统一为美元（USD）**：全文 2,000 万 / 5,000 万均指美元。
2. **大币种补保底数字**：否则该项无法考核、无法判断达成。
3. **客户指标换口径**：用 **"产生交易量的有效租户数"** 替代"意向 / 签约数"，更贴合平台模式。
4. **每个项目绑定先行里程碑**：上线时间 → 渠道就绪 → 交易量，三段式可追踪。
5. **设保底 / 挑战双目标**：交易量给出"保底 + 挑战"两档，既稳妥又有牵引。
6. **风险 0 事件作为硬性底线**：渠道 / 资金风险事件一旦发生，对应渠道目标视为未达成。
7. **团队建设与业务量强绑定**：明确"先有量、再补人"，呼应成本可控的整体策略。
8. **把"价格"作为显性经营杠杆**：法币租户对价格敏感，H2 用适度降价换进件与交易量，但要框定降价幅度与边界、并测算对 take rate 的影响。
