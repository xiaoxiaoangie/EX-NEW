EX White Label — web2白标门户解决方案

即开即用，无需开发
文档类型: 白标解决方案指南 版本: v1.0 最后更新: 2026-04-12 配套: EX Open API 解决方案

1方案概述

EX 根据客户开通的产品线，由运营团队完成配置后，直接交付两个门户：

门户 英文 使用方 说明
租户门户 Tenant Portal（TP） 客户（机构） 客户自己使用的运营管理后台，管理商户、查看交易、配置产品等
商户门户 Merchant Portal（MP） 客户的终端用户（商户） 客户交付给其终端商户使用的操作门户，商户自助完成收付款、发卡等操作

│  │   运营配置完成     │ →  │   交付 TP + MP    │               │
│  │ (产品开通+品牌配置)│    │  (即开即用)       │               │

└──────────────────┘     └──────────────────┘
核心价值：客户无需任何开发投入，开通产品后即可获得完整的运营管理后台（TP）和面向终端商户的操作门户（MP），快速启动业务。

2谁适合白标方案

客户类型 场景 推荐理由
代理机构 代理 EX 产品给终端商户 无需开发，TP 管理商户，MP 交付商户
跨境电商服务商 为卖家提供收付款服务 快速上线，卖家直接用 MP 操作
初创 Fintech 刚起步，优先验证业务模型 零开发成本，先跑通业务再考虑 API 集成
传统贸易公司 需要数字化收付工具但无技术团队 门户即开即用，降低数字化门槛
持牌金融机构 拓展新业务线（如发卡、加密） TP 管理，快速扩展产品线

3白标门户能力矩阵

根据客户开通的产品线，TP 和 MP 自动展示对应的功能模块：

TP（租户门户）— 客户使用

功能模块 说明 备注
商户管理 注册商户、商户基本信息提交与跟踪、商户状态管理
商户开通产品查询 查看商户开通的产品；商户增开产品预计6月上线，目前只能开单一产品 针对SP的要求，目前分为B2B的业务、B2C的业务（Shopee、TK、Amazon等）；如果客户开通了一个B2B的业务，再开通B2C的业务，需要找运营团队支持
商户收费管理 管理商户产品收费 对客的标准报价，需要发邮件给运营；如果商户要单独报价，可以进行单独收费配置；需要有一个人配置，一个人审核
商户汇率报价 管理商户的汇率加点 同样，默认统一的加点配置，可以发运营；商户特殊配置在这里配置
商户交易管理 查看所有商户的收款、付款、换汇 按产品线展示
商户贸易背景材料 商户的外贸订单，店铺管理
工单审核 费率，汇率的审核人员进行审核，审核通过后才能生效
我的分佣 您获取的交易分佣账单和结算单 所有产品
增值功能：销售管理 给每个销售生成一个拓客链接
增值功能：代理商管理 可以发展下一级的代理商 目前只有交易可以返点，汇率的返点暂不支持；操作手册可以跟运营获取
账号权限 可以给其他人分配账号 目前权限只能控制到页面，不能控制到操作

MP（商户门户）— 终端商户使用

功能模块 说明 备注
商户注册+入网 商户开通比如外贸收款，提交外贸收款的需要的资料，包括实名认证的信息和业务信息 这基于SP能做的业务
法币收款 申请 VA、查看收款记录、下载凭证 会区分B2B的贸易收款、B2C的电商等场景收款
法币付款 添加收款人、发起付款、结汇人民币、查看付款记录 需要先添加收款人，再发起付款
换汇 查询汇率、发起换汇 目前是实时汇率产品
贸易单据 上传/管理贸易背景文件 法币产品
店铺管理 管理自己的电商平台店铺等

4交付流程

├── 1. 线下签约
│     └── 跟客户经理签订合同
│
├── 2. EX 运营配置（1-2 个工作日）
│     ├── 创建租户（Tenant）
│     ├── 配置品牌信息（Logo、域名、邮件模板）
│     └── 配置商户的标准报价/汇率
│
├── 3. 交付 TP（租户门户）
│     └── 客户获得 TP 管理后台访问权限
│     └── 客户可开始在 TP 中注册和管理商户
│
├── 4. 交付 MP（商户门户）
│     └── 商户通过租户的链接注册
│     └── 完成 KYC/KYB → 产品开通 → 开始使用
│
└── 5. 上线运营
      └── 客户通过 TP 管理日常运营
      └── 终端商户通过 MP 自助操作

交付时间线

阶段 内容 预计耗时
线下签约 签订合同 1-2 天
运营配置 租户创建、费率配置，域名配置等 1-2 个工作日
交付 交付MP 和TP 1天（含在上面的时间，配置完成就交付）
生产验证 商户注册 → KYC → 产品开通 3-5 个工作日（含审核）
总计 从签约到首批商户可操作 2-10工作日（看生产验证的情况）
