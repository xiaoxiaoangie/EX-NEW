## 二、客户痛点 — 自建发卡系统，每一步都是坑

做过发卡的都知道，发卡系统从来不是"写个后台"那么简单。我们见过太多客户在以下环节踩坑、返工、甚至被卡组关停：

| # | 痛点                          | 自建的真实代价                                                                                                      |
| - | ----------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| 1 | **对接链路复杂**        | Processor、卡组（Visa/Mastercard）、BIN 赞助方、Issuer 各家标准不一，协议、报文、清算口径全不同，光摸清规则就要数月 |
| 2 | **周期长、成本高**      | 从零搭建客户端 + 运营后台 + 风控 + 对接，自建团队普遍 6–12 个月起步，且人力成本居高不下                            |
| 3 | **合规风控是生死线**    | KYC/KYB/KYT/AML 做不到位，轻则被卡组警告，重则直接关停 BIN —— 这是压垮发卡业务最常见的原因                        |
| 4 | **资金与清算混乱**      | 多渠道备付金管理、对账差错、客户余额与渠道余额脱节，财务对不上账                                                    |
| 5 | **数币 + 法币双轨**     | 既要支持加密货币充值，又要法币发卡消费，技术栈横跨两个世界，自建团队往往力不从心                                    |
| 6 | **虚拟卡 + 实体卡并行** | 卡产品形态多、卡 BIN 多、生命周期管理复杂，缺少成熟的卡管理体系                                                     |
| 7 | **运营效率低**          | 没有体系化的客户管理、计费收费、渠道路由后台，业务全靠人肉操作，规模一上来就崩                                      |
| 8 | **找不到对的团队**      | 既懂支付、又懂发卡、还有真实上线经验的团队极少 —— 大多数开发商是"第一次做发卡"                                    |

> **一句话总结**：发卡系统的难点不在"能不能做出来"，而在"能不能合规地、稳定地、规模化地跑起来"。

## 三、我们的方案 — 一套交钥匙的 AI 智能发卡 SaaS 平台

我们提供的不是单点功能，而是一套**端到端、可直接承载真实交易**的发卡平台。覆盖发卡业务的全链路：**向上服务客户，向下对接渠道，中间做好运营、计费与风控。**

### 3.2 四大核心能力

**1）客户端（Merchant Portal）— 让你的客户自助跑起来**

- 注册登录、KYC/KYB 资料提交、安全设置（2FA）
- 多币种充值（法币 / 数币）、余额查询、资金流水
- 开卡申请、卡列表、卡详情与消费记录、卡充值、冻结/解冻/注销
- 交易查询与导出、账单查看、费率展示
- API Key 与 Webhook 自助管理，支持客户对接自有系统

**2）运营后台（Operation Portal）— 让你的团队高效运营**

- **客户管理**：客户档案、产品配置、账户管理（止入止出/冻结）、交易管理、入网审核工作流
- **渠道管理**：渠道接入配置、实时余额与预警、渠道对账、多渠道智能路由、渠道成本核算
- **计费收费**：费率方案、个性化/阶梯定价、账单自动生成、收费流水、分润结算
- **管理后台**：用户与角色权限（管理员/财务/运营/风控）、操作审计、系统配置、数据看板

**3）风控合规系统（独立可选模块）— 守住业务生死线**

- **KYC**：证件 OCR、人脸活体、制裁名单 / PEP 筛查
- **KYB**：企业核验、UBO 识别、负面信息筛查、多级审核
- **KYT**：实时风控规则引擎、异常交易识别、AML 监控、可疑交易上报
- **交易策略**：限额/频次/地域/MCC 自定义、多级审批、黑白名单、灰度发布
- **合规运营**：RFI 工作流、案件处置、风险等级复评、监管报告

**4）渠道对接层 — 把最难的脏活累活交给我们**

- Processor 对接、卡组（Visa / Mastercard）对接、Issuer / BIN 赞助方对接
- 美国卡 BIN、欧洲 BIN、香港卡组等多区域通道，已有成熟落地经验
- 数币 + 法币双轨发卡，虚拟卡 + 实体卡全形态支持
