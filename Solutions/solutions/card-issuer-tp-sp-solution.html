1业务概况

维度说明
行业广告代理 — Google 一代 / Facebook 国内二代
现有能力自有发卡系统，前端有客户（广告主），后端对接发卡渠道
平台定位一站式发卡管理平台 — 向上管理广告主客户，向下对接发卡渠道
核心诉求需要一套管理平台，包含客户端（广告主自助使用）和操作后台（内部运营管理）

3管理后台功能清单

👥客户管理
  • 客户基本信息（KYC/KYB 资料、营业信息、联系方式）
  • 客户产品配置（开通/关闭产品线、产品参数设置）
  • 客户账户管理（查余额、止入、止出、冻结/解冻）
  • 客户交易管理（交易列表、交易详情、交易状态查询）
💰计费收费
  • 费率方案管理（创建/编辑/上下架费率模板）
  • 客户费率配置（按客户绑定费率方案、支持个性化定价）
  • 账单生成与查看（按周期自动生成、支持导出）
  • 收费记录查询（手续费明细、扣费流水）
🔗渠道管理
  • 渠道基本信息（渠道名称、接口配置、状态管理）
  • 渠道余额（实时余额查询、余额预警）
  • 渠道交易管理（交易列表、交易明细、对账）
  • 渠道路由配置（优先级、分流规则）
⚙️管理后台
  • 用户管理（创建/编辑/停用用户账号）
  • 权限管理（角色定义、菜单权限、数据权限、操作权限）
  • 操作日志（关键操作审计记录）
  • 系统配置（通知配置、安全设置）
🛡️合规风控系统独立模块 · 可选开通
KYC 个人身份验证
  • 身份证件采集与 OCR 识别
  • 人脸识别 / 活体检测
  • 制裁名单筛查（Sanctions）
  • PEP（政治敏感人物）筛查
KYB 企业尽调
  • 企业注册信息核验
  • UBO（最终受益人）识别
  • 营业执照 / 公司章程采集
  • 企业制裁与负面信息筛查
KYT 交易监控
  • 实时交易风控规则引擎
  • 异常交易模式识别
  • AML 反洗钱监控
  • 可疑交易自动预警与上报
交易策略配置
  • 风控规则自定义（限额、频次、地域、时段等）
  • 多级审批流程（自动通过 / 人工复核 / 自动拒绝）
  • 黑白名单管理（客户、卡号、IP、设备）
  • 风控策略灰度发布与 A/B 测试
合规运营
  • RFI（补充材料请求）工作流
  • 合规案件管理与处置
  • 定期客户风险等级复评
  • 监管报告生成与导出

5核心业务流程

1客户入网 → 产品开通 → 开卡消费
广告主申请→ 平台审核→ 开通产品→ 客户充值→ 渠道发卡→ 广告消费
2计费流程
客户交易→ 计费引擎→ 手续费计算→ 自动扣费→ 账单生成
基于客户绑定的费率方案，每笔交易自动计算手续费并从客户账户扣除
3渠道资金流
平台预充值→ 渠道账户→ 发卡/充卡→ 客户卡消费→ 渠道扣款
平台向渠道预充值备付金，客户开卡/消费从渠道余额扣减，需关注渠道余额预警

7补充说明

项目说明
合规风控可选附加模块。如需 KYC/KYB 审核、AML 监控等合规能力，可额外开通
API 对接平台提供标准 RESTful API + Webhook，现有发卡系统可通过 API 与管理平台集成
部署模式SaaS 模式，通过 Web 管理后台操作，无需自建基础设施
数据迁移如有存量客户和渠道数据，需评估迁移方案（范围待确认）
客户端形式支持 Web 端、后续可扩展移动端（H5 / App）
⚠️ 排期为预估，实际交付时间取决于需求细化程度、渠道联调复杂度和双方资源投入。建议先完成 Phase 0 需求确认后，细化各阶段排期。
