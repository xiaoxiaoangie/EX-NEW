## 〇、执行摘要（TL;DR）

1. **本地支付是「账户 + QR」统治的市场，不是卡组织市场。** NAPAS / VietQR 近乎零成本、实时到账，已成国民习惯；外卡、VCC、外国钱包（Alipay 等）体验割裂——**这是结构性缺口，也是机会**。【公开 + 一手】
2. **越南是全球加密采用率第一梯队**（约 **2,100 万**持币人群），但官方零售合规通道几乎为零，绝大多数走 **OTC / P2P + 黑市汇率**，USDT 事实上充当「美元替代 + 跨境结算媒介」。【公开 + 一手】
3. **2025 年起密集立法**：6 月《数字技术产业法》首次在法律层面承认数字资产为财产（但**不承认为支付手段**）；9 月 Resolution 05 启动 **5 年加密试点**；同期 Resolution 222 设立**胡志明市 + 岘港两个国际金融中心（IFC）**。【公开】
4. **持牌交易所门槛极高**：最低注册资本 **10 万亿越南盾（≈3.78 亿美元）**、外资 ≤49%、机构股东 ≥65%、全部以 VND 计价、Level 4 安全认证；落地极难。【公开】
5. **现实落地窗口是岘港 VIFC 沙盒**（Basal Pay、MIMO、9Pay 等），但**进展慢、范围窄（仅岘港 3 区）、沙盒本身难盈利**，业内普遍选择「**离岸 VASP + 越南持牌 PSP 合同**」的折中架构。【公开 + 一手】
6. **监管执法趋严**：2026 年 1 月起「**从 OffRamp 开始监管**」；冻卡、关停线下 OTC、生物识别清账户同步进行。「合规 + 不动卡 + 可追溯」本身成为卖点。【公开 + 一手】
7. **真实需求三条线**：转口/出口贸易收款、跨境电商收单、稳定币/侨汇与资金出境——都被现有合规渠道压抑，外溢到 OTC 与加密。【一手】

## 一、本地支付现状

### 1.1 NAPAS / VietQR：统治级的「账户直转」

| 项                     | 内容                                                                                                                                                                                | 来源            |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------- |
| **NAPAS**        | 越南国家级跨行清算与转接网络（National Payment Corporation of Vietnam），本地「银联」，小额转账基本零费                                                                             | 【公开 + 一手】 |
| **VietQR**       | NAPAS 于**2021 年**推出的统一二维码标准，扫码即时到账、可互通，覆盖几乎所有银行与商户                                                                                         | 【公开+一手】   |
| **实时清算轨道** | NAPAS、VietQR、CITAD；监管方为越南央行 SBV                                                                                                                                          | 【公开】        |
| **使用习惯**     | A2A（账户到账户）转账近乎零费、秒到，**已远比卡支付普及**；连小餐馆都用个人 QR。习惯上「小额扫 QR、大额走银行转账」，**但二者本质都是银行账户直转**（详见下方轨道说明） | 【公开 + 一手】 |

> **核心含义**：越南是「账户 / QR 直转」主导市场，**不是卡组织主导**——这从根本上决定了外卡、钱包 在越南天然水土不服。任何想进越南本地收付的玩家，**必接 VietQR / NAPAS**。

> **轨道澄清（重要）**：**NAPAS 本身就是一种银行账户转账方式（账户直转）**，VietQR 只是跨在 NAPAS 清算轨道上的**扫码发起入口**。因此 **QR 与即时转账同轨、共用同一套限额（不按 QR / 转账分别设限）**：
>
> - **小额（单笔 < 5 亿 VND）** → 走 **NAPAS 247 即时清算**（秒到、零费），无论扫 QR 还是输账号转账都一样；
> - **大额（单笔 ≥ 5 亿 VND）** → **这就是所谓「大额银行转账」**，超出 NAPAS 247 即时上限，**转走常规清算（IBPS / CITAD）**，到账 ≥ 4 小时或次工作日，并需 AML 上报（见 §1.1.1）。

#### 1.1.1 NAPAS247 / VietQR 转账：费率·时效·限额（核实）

| 维度                         | 实际参数                                                                                                                                                                           | 来源            |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------- |
| **时效**               | **NAPAS 247 全天候 24/7 实时到账**，无截止时点，通常数秒内到账                                                                                                               | 【公开】        |
| **单笔限额**           | **单笔 < 5 亿越南盾（约 1.9~2 万美元）** 才走 NAPAS 即时清算；HSBC 等列明扫码 / 即时转账单笔区间 **2,000 ~ 499,999,999 VND**                                           | 【公开】        |
| **大额（≥5 亿 VND）** | **2025-11 修订通函**：央行电子清算系统单笔上限 5 亿 VND，**超过须走常规清算（IBPS/CITAD）**，到账 **≥4 小时或次工作日**；**15:55 后发起 → 次工作日**处理 | 【公开】        |
| **「拆单」被叫停**     | 此前银行会把大额拆成多笔 <5 亿 VND 以走即时通道（部分收约 1 万 VND/笔）；MSB、Eximbank、TPBank、VPBank 等自 2025-05 起**陆续停止拆单**                                       | 【公开】        |
| **费率**               | 多数银行**免收**个人即时转账费（零费）；**QR 收款费率显著低于卡**，建设 / 接入成本低                                                                                   | 【公开 + 一手】 |
| **生物识别门槛**       | **单笔 > 1,000 万 VND（约 379 美元）须人脸验证**（2025-01 起，见 §3.4）                                                                                                     | 【公开】        |
| **网络覆盖**           | NAPAS 成员约**68 家**银行 / 金融机构；约 2.06 万台 ATM、约 74.1 万台 POS/mPOS                                                                                                | 【公开】        |
| **大额上报**           | **境内转账 ≥5 亿 VND 须上报** SBV 反洗钱部门（Circular 27/2025，见 §3.4）                                                                                                  | 【公开】        |

> **关键洞察**：**5 亿越南盾（约 1.9~2 万美元）是一条贯穿性红线**——它既是 ① NAPAS 即时清算的单笔上限、又是 ② AML 强制上报阈值。两条线叠加意味着：**小额（<5 亿且 >1,000 万需刷脸）走即时零费 QR / A2A 极顺畅；大额则即时性下降、可追溯性陡增、拆单规避已被封堵**。这正是越南本地收付「**小额极致顺滑、大额受控**」的结构特征，也直接影响 OTC / OffRamp 的单笔设计与拆分策略。
>
> 注：表中限额/门槛 **不区分 QR 还是输账号转账**——VietQR 只是 NAPAS 247 同一轨道上的扫码发起方式；「大额银行转账」指的就是 **≥ 5 亿 VND 转常规清算（IBPS / CITAD）** 那种。

### 1.4 跨境 QR 互通：方向对、进展慢、反复试

> 用户判断：**跨境 QR 互通这件事非常慢**，反复试错、难一步到位。下述事实与此一致。

**一、云闪付（UnionPay / QuickPass）早期尝试：发布当天能用、随后即停**

- 一手访谈：某本地合作方曾与「收钱吧」推云闪付，已与 **NAPAS、VMK（本地银行）** 谈好合作、POS 系统也开发完成；**但召开新闻发布会后系统「只能用了 2 天」便停掉**（与“发布当天还行、过了当天直接用不了”的说法一致）。【一手】
- 退出原因（访谈推测）：云闪付打通微信 / 支付宝 / 银联后放弃。【一手】

**二、最新：NAPAS × UnionPay 中越跨境 QR 试点（分阶段推进）**

- **NAPAS × UnionPay International（联合 ICBC、Vietcombank）** 启动中越跨境 QR 试点：**第一阶段仅支持中国游客**在越南扫商户 **VietQR Global** 码付款（Central Retail 超市等首批接入）；**越南人赴华扫码**预计 **2026 年初**才上线。【公开】
- **2025-04，NAPAS × Ant International（蚂蚁）× Vietcombank** 也在推 Alipay 扫 VietQR Global。【公开】
- 方向上 NAPAS 正把 VietQR 与中国（Alipay/微信 / 银联系）、韩国、泰国、柬埔寨、老挝等双向打通。【公开】

> **解读**：跨境 QR 互通 **“方向对、但落地慢”**——早年云闪付是“发布一阵子就用不了”；即使是现在的 NAPAS×UnionPay / Ant 试点，也是**单向（先只允许入境游客付）、分阶段、依赖头部银行 / 卡组织逐个打通**。对依赖跨境 QR 的业务来说，**不能假设互通“一步到位”**。

### 2.4 越南全面战略合作伙伴关系版图（宏观背景）

> **资料来源**：**开泰研究中心（KResearch Center）**——泰国开泰银行（Kasikornbank）旗下研究机构。下表为其整理的「越南将双边关系升级为全面战略合作伙伴关系，以在不同方面支持越南经济发展目标」。【公开】

| 与越南建立全面战略合作伙伴关系的国家 / 地区 | 对越南的重要性                            |
| ------------------------------------------- | ----------------------------------------- |
| 美国、欧盟、英国、法国                      | **高科技产品出口市场**              |
| 中国、韩国、日本、新加坡、马来西亚和泰国    | **外商直接投资（FDI）及供应链投资** |
| 俄罗斯、澳大利亚、欧盟                      | **能源和关键矿产安全**              |
| 澳大利亚、新西兰                            | **绿色经济及食品标准提升**          |
| 新加坡和英国                                | **金融业投资**                      |
| 中国、美国、俄罗斯、印度和印度尼西亚        | **国际安全及增强地缘政治博弈权力**  |

**支付 / 跨境视角分析**：

- **出口市场（美 / 欧 / 英 / 法）= 收款侧需求来源**：越南高科技 / 制造出口主要面向欧美 → 出口商对 **USD/EUR/GBP 多国 VA 收款 + 结汇 VND** 的需求强，与 §2.2 的「瓶颈在海外收款侧」相互印证。
- **FDI / 供应链（中日韩新马泰）= 资金流入与 B2B 结算**：中国是供应链投资与转口贸易主源头，对应大量「中国商户 → 越南买家」的跨境收付与 USDT 结算需求（§二、§七）。
- **金融业投资（新加坡、英国）= 合规架构落点**：越南金融开放重点引入新加坡、英国资本，呼应业内「**离岸（新加坡 MPI / 香港）主体 + 越南本地落地**」的折中架构（§七、§八）。
- **能源 / 关键矿产、绿色经济**：与支付主业关联弱，主要影响大宗贸易结算与 ESG 合规，属外溢机会。

> **解读**：该版图说明越南的对外经济战略是「**欧美做出口市场、东亚做投资与供应链、新英做金融**」的分工结构。对支付玩家的含义——**收款腿（欧美）与投资 / 结算腿（中日韩新）天然分离**，恰好对应「多国 VA 收款 + 结汇 VND + 跨境 B2B 结算」的产品组合需求。

| **核心痛点**     | **冻卡风险**——「钱不干净」会被冻结，外贸客户挣 10% 被冻一次就受不了 | 【一手】 |

### 3.2 黑市 / 换汇的资金流转方式

> 以下为**业内普遍做法的推断性描述**（综合一手访谈 + P2P 公开机制），并非单一权威来源，仅供理解资金流向。

| 金额                  | 典型流转方式                                                                              | 逻辑                                                                                                                          | 来源                    |
| --------------------- | ----------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- | ----------------------- |
| **小额**        | **个人卡 / 个人账户互转**（A2A）、个人 VietQR 扫码                                  | 越南 A2A 转账近乎零费秒到、人人用个人 QR；小额拆分在个人账户间流转，**不易触发大额上报阈值**，混入日常转账噪音          | 【公开 + 一手（推断）】 |
| **大额**        | **银行转账**（个人/公司账户），或拆成多笔                                           | 大额仍走银行轨道效率最高；但**2025-09 起单笔 ≥5 亿越南盾（约 1.9 万美元）须上报** SBV 反洗钱部门，大额可追溯性大幅上升 | 【公开 + 一手（推断）】 |
| **跨境 / 换汇** | 地下钱庄「两地对敲」（境内收 VND、境外付 RMB/USD，不真正跨境）；或**USDT 作跨境腿** | 规避外汇管制；USDT 充当价值转移媒介，OTC 在两端分别用本地账户收付法币                                                         | 【公开 + 一手】         |
| **加密 P2P**    | 币安/Bybit P2P 用**个人 VND 银行转账**买卖 USDT，平台托管担保；大户走私下群撮合     | 无本地持牌交易所 / 法币 onramp，P2P 成为事实基础设施                                                                          | 【公开】                |

> **核心机制**：无论小额个人卡互转还是大额银行转账，**法币侧都依赖个人 / 公司银行账户 + NAPAS 轨道**；这正是「冻卡」成为最大痛点的根源——一旦某账户被标记，关联资金链受牵连。

### 3.3 现金为何仍然很多

| 成因                            | 说明                                                                      | 来源     |
| ------------------------------- | ------------------------------------------------------------------------- | -------- |
| **税务规避**              | 本地商家用现金规避增值税 / 所得税；                                       | 【一手】 |
| **资金安全 / 不信任银行** | 怕货币贬值、怕银行倒闭，偏好现金 + 黄金计价（黄金买房根深蒂固）           | 【一手】 |
| **规避监管 / 冻卡**       | 银行流水可被追踪冻结；现金「无记录」，本地员工也不愿替中国人存现金 / 转账 | 【一手】 |
| **现金习惯**              | 长尾零售、城乡差距，现金仍是大量日常支付场景的默认选项                    | 【一手】 |

> 含义：现金需求的根因是**税 + 资金安全 + 规避监管**三重叠加。访谈者视「现金收单 / 现金前置通道」为潜在蓝海，但持牌机构一般不碰，**合规风险极高**。

### 4.2 加密钱包 × NAPAS：软性 On/Off-Ramp

- 越南不少加密钱包已对接 NAPAS 做本地 **payout（下发）/ collection（收款）**——用户的链上 / 稳定币余额可直接经 NAPAS/VietQR 完成 VND 收付。【一手】
- **机制本质**：「钱包侧持币、NAPAS 侧落 VND」，把 OTC 换汇动作**隐藏进收付流程**——这是一种「软性 On/Off-Ramp 嵌入支付」。【一手】
- 含义：**「本地收付 + NAPAS 接入」是越南数币钱包能否被日常使用的准入门槛**，而非加分项。【一手】

## 七、On/Off-Ramp 与 OTC 现状

### 7.1 监管方向

- **2026 年 1 月起「从 OffRamp 开始监管」**：政府宣布监管 Web3 业务，先从法币↔数币的出金环节切入。【公开 + 一手】
- **OffRamp 合规试点先行者是 MIMO（Dragon Lab JSC），不是 MoMo**：MIMO 是岘港沙盒下**首个**获受控试点许可的 USDT↔VND 非托管兑换方案（由岘港科技厅发证，非 SBV 支付牌照）。**MoMo 是另一回事**——它是越南最大的电子钱包（持牌支付 / 钱包主体，仅限本地人），与 MIMO 无关，二者不应混淆。【公开 + 一手】
- **谁与 MIMO / Dragon Lab 合作**：**9Pay（9Pay JSC，即纪要中的「南配 / Nanpei」）** 借 MIMO 做 OffRamp 落地；Dragon Lab 另有分支 **Gaia**。注意：**9Pay 与 Dragon Lab 仍是两家独立公司**（是合作关系，不是同一主体）。**Holdstation 与 MIMO 无合作**。【公开 + 一手】
  > ⚠️ 注：一手会议曾出现「MOMO 是唯一持牌 OffRamp」的说法，经核对**应为 MIMO 的口误 / 转录混淆**——MIMO（兑换方案）与 MoMo（电子钱包）是完全不同的两个主体。
  >
- **稳定币 B2B 直接结算在越南不合法**：合规玩家普遍通过离岸实体走 **USDT → USD → VND** 路径规避，并对收款方做 KYB。【一手】

### 7.2 OTC 的真实运作（产品需求来源）

### 7.4 KYC / KYB 要求（访谈样本）

- **公司**：KYB（公司名、注册文件、UBO ≥25%）；**个人**：KYC（收款方）。【一手】
- 多数服务商**只需业务合同（business agreement）**，不强制要求海关完税证明（customs declaration）——因中越多陆运、商户常无报关。【一手】
- 一次性 KYB，后续同账户免重复 KYC。【一手】
