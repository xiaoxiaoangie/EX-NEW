资金流向总览 · 3 大场景

钱包充值 / 市场消费 / 提现提币
场景一：钱包充值（OffRamp 式）
C 端 USDT → 链上 → SP 收单地址

→ USD 到传音 SP 端 USD 账户

报价不锁定：EX 返回实时汇率仅作展示，最终以 SP 成交为准。建议传音前端 5-10 分钟刷新一次。

场景一 · 钱包充值业务流程

数币 → 实时询价 → autoOffRamp → 法币上账

关键步骤

  1. C 端在传音 App 发起"充值 100 USD"
  2. 传音调 POST /offramp/quote 取实时汇率（仅展示，不锁价），前端每 5-10 分钟刷新
  3. 传音展示"当前汇率约需 100.X USDT" → C 端确认
  4. 传音调 POST /cashier/orders 带 autoOffRamp:true，取收款地址
  5. C 端链上转账 → CRYPTO_PAYIN_SUCCESS
  6. EX 自动触发 OffRamp（按实际成交实时汇率）
  7. USD 入账传音 SP 账户 → OFFRAMP_SETTLED 含 actualFiatAmount
  8. 传音按 actualFiatAmount 给 C 端钱包上账

场景三 · 传音提现 / 提币

法币 & 数币分别出金

USD 提现（→ 银行账户）

  1. 传音添加银行收款人 → Webhook 返回审核结果
  2. 传音调 POST /payout/orders 发起 USD 提现
  3. EX 回调 PAYOUT_PROCESSING
  4. SP 完成付款 → EX 回调 PAYOUT_SUCCESS / FAIL

USDT 提币（→ 链上地址）

  1. 传音添加链上收款地址 → Webhook 返回审核结果
  2. 传音调 POST /crypto/withdraw 发起提币
  3. EX 回调 CRYPTO_WITHDRAW_PROCESSING
  4. 链上确认 → EX 回调 CRYPTO_WITHDRAW_SUCCESS / FAIL
⚠️
报价不锁定：EX 返回实时汇率仅作展示，最终以 SP 成交为准。建议传音前端 5-10 分钟刷新一次。
