# 代收付主体方案

> **版本**：v1.1 | **日期**：2026-05-05 | **机密等级**：内部

---

## 1. 背景与痛点

### 1.1 核心矛盾：瑞士 SRO 合规要求 vs 业务拓展需求

我们持有 **瑞士 SRO（自律监管组织）颁发的金融中介牌照**。SRO 对持牌机构的展业行为有严格约束：

| SRO 合规约束 | 具体表现 | 对业务的影响 |
|-------------|---------|-------------|
| **客户准入标准** | SRO 要求严格的 KYC/KYB，部分高风险地区客户直接拒绝 | 大量潜在客户无法以持牌主体直接承接 |
| **交易类型限制** | 对交易对手、资金来源、交易用途有明确白名单/黑名单 | 部分合法但高敏感的交易场景无法通过合规审核 |
| **资金用途审查** | 每笔资金需提供明确用途说明，大额交易需额外审批 | 审批效率低，客户体验差，业务流失 |
| **定期审计与报告** | SRO 定期检查，任何异常交易可能导致牌照风险 | 持牌主体必须保持"干净"的交易记录 |

**核心矛盾**：严格按 SRO 要求展业 → 业务规模受限；放松标准 → 牌照风险。

### 1.2 落地市场的本地合规要求

除了 SRO 层面的约束，各目标市场还有本地化的合规要件：

| 市场 | 本地合规要求 | 为什么需要代收付主体 |
|------|------------|-------------------|
| **越南** | 进口交易需本地企业报关单；银行入账要报关单号；需开红色发票 | 必须有越南本地实体才能取得报关单和发票 |
| **香港** | 金钱服务经营者牌照（MSO）门槛相对低；银行开户对资金来源要求严格 | 适合作为亚太区资金归集中心 |
| **新加坡** | 支付服务法（PSA）下需 MPI/SPI 牌照；但贸易公司不受 PSA 管辖 | 以贸易公司身份可规避 PSA 牌照要求 |

### 1.3 核心需求

设立 **"代收付主体"** —— 在特定法域注册的合法实体，**与瑞士持牌主体保持股权和业务隔离**：

- 与买方签订采购/服务合同 → 合法收款依据
- 与卖方签订供货/服务合同 → 合法付款依据
- 提供当地合规所需文件（报关单、发票、税务凭证等）
- **风险隔离** → 持牌主体不直接面对 SRO 敏感的交易类型
- **牌照保护** → 代收付主体承接的业务不纳入 SRO 审计范围

## 2. 代收付主体的角色定位

### 交易结构

```
买方 (Buyer)                                         卖方 (Seller)
   │                                                     ▲
   │ ① 采购合同 + 付款                        ⑤ 供货合同 + 付款 │
   ▼                                                     │
┌────────────────────────────────────────────────────────────┐
│                代收付主体（HK / SG / VN）                     │
│  · 当地注册企业              · 与买卖双方分别签订合同            │
│  · 持有合规文件(报关单等)      · 资金在代收付主体账户中转          │
│  · 与瑞士持牌主体无股权关联    · 以贸易/服务公司身份运营          │
└──────────────────────────┬─────────────────────────────────┘
                           │
                 ③ 资金归集 │ ④ 结算（独立商业合同）
                           ▼
              ┌──────────────────────────┐
              │  瑞士 SRO 持牌主体         │
              │  （仅处理合规筛选后的资金流） │
              └──────────────────────────┘
```

### 核心职能

| 职能 | 说明 |
|------|------|
| **合同载体** | 与买方签采购合同、与卖方签供货合同，形成完整贸易链条 |
| **合规文件持有** | 持有当地法律要求的文件：进口报关单、商业发票、税务凭证等 |
| **资金中转** | 收到买方款项后按合同约定付给卖方，差额为服务费/佣金 |
| **风险隔离** | 高合规风险交易隔离在代收付主体内，不直接关联持牌主体 |
| **本地化桥梁** | 满足当地监管对"本地实体"的硬性要求 |

## 3. 法域选择：香港 vs 新加坡 vs 越南

### 3.1 设立代收付主体的首要目标

**核心目标：让瑞士 SRO 持牌主体保持"干净"**

代收付主体的设立不是为了逃避监管，而是为了：
1. 将 SRO 敏感交易（高风险地区客户、复杂贸易背景）放到独立主体处理
2. 经过代收付主体初筛后的合规资金，再进入持牌主体体系
3. 持牌主体账上只留审计友好的、有完整合规链条的交易

### 3.2 三大法域对比

| 维度 | 🇭🇰 香港 | 🇸🇬 新加坡 | 🇻🇳 越南 |
|------|---------|----------|---------|
| **核心定位** | 亚太资金归集中心 | 合规高地 + 东南亚枢纽 | 落地市场本地主体 |
| **设立形式** | 香港有限公司 | 新加坡私人有限公司 (Pte Ltd) | 越南有限责任公司 (TNHH) |
| **与瑞士SRO的关系** | 独立法域，瑞士SRO管辖不到 | 独立法域，MAS与FINMA无直接联动 | 独立法域，越南监管与瑞士无关 |
| **金融牌照要求** | 不从事汇款/支付 → 不需MSO牌照 | 不从事支付服务 → 不需PSA牌照 | 不从事金融 → 不需牌照 |
| **经营范围定位** | 国际贸易、商务咨询、代理服务 | 国际贸易、商务咨询、代理服务 | 进出口代理、国际贸易、咨询 |
| **银行开户难度** | ⭐⭐⭐ 中等（需说明业务模型） | ⭐⭐⭐⭐ 较难（银行审查严） | ⭐⭐ 较易（外资企业常规流程） |
| **开户周期** | 4-8周 | 6-12周 | 4-8周 |
| **可开多币种账户** | ✅ USD/EUR/HKD/CNY等 | ✅ USD/EUR/SGD等 | ✅ USD/EUR + VND |
| **外汇管制** | 无外汇管制，资金自由进出 | 无外汇管制，资金自由进出 | 有外汇管制，跨境收付需银行审核 |
| **当地文件获取** | ❌ 无报关单需求场景 | ❌ 无报关单需求场景 | ✅ 可取得进口报关单、红色发票 |
| **企业所得税** | 8.25%（首200万HKD）/ 16.5% | 17%（部分减免） | 20% |
| **无须审计** | 需年审，但不强制审计(小公司) | 小公司可豁免审计 | 外资企业必须年度审计 |
| **国际声誉** | ⭐⭐⭐⭐⭐ 全球认可 | ⭐⭐⭐⭐⭐ 全球认可 | ⭐⭐⭐ 新兴市场 |
| **人员成本** | 高 | 高 | 低 |
| **设立成本（含开户）** | 5,000-10,000 USD | 8,000-15,000 USD | 5,500-9,000 USD |
| **年运营成本** | 30,000-60,000 USD | 50,000-80,000 USD | 49,000-82,000 USD |

### 3.3 各法域适用场景

#### 🇭🇰 香港 — 推荐作为**亚太资金归集中心**

**适合场景**：
- 承接亚太区客户的收付款合同签署
- 资金归集后统一结算到瑞士持牌主体
- 不需要本地报关单等特殊文件的交易

**核心优势**：
- **无外汇管制**：资金自由进出，无需审批
- **税率低**：利得税 8.25%/16.5%，离岸收入可申请豁免
- **银行选择多**：汇丰、恒生、星展、众安等，多币种账户齐全
- **国际认可度高**：买卖双方签约接受度好
- **与瑞士SRO完全隔离**：香港公司不受FINMA/SRO管辖

**关键注意**：
- 银行对贸易公司频繁大额资金进出会审查
- 需向银行提供清晰的商业模式说明
- 建议多银行开户分散风险

#### 🇸🇬 新加坡 — 推荐作为**合规高地 + 东南亚枢纽**

**适合场景**：
- 面向东南亚（印尼、菲律宾、泰国）的交易合同签署
- 需要高合规声誉背书的交易场景
- 涉及数字资产/加密相关的贸易代理

**核心优势**：
- **合规声誉极高**：新加坡公司签署的合同银行信任度最高
- **PSA不管贸易公司**：只要不从事"支付服务"（为他人转移资金），不需PSA牌照
- **数字资产友好**：涉及加密资产的贸易代理，新加坡接受度高于香港
- **与瑞士SRO完全隔离**：MAS与FINMA无跨境联动协议

**关键注意**：
- 银行开户最严格，审查周期长（6-12周）
- 需要证明有实质性经营（substance）
- 建议有本地董事 + 实际办公地址
- 运营成本高于香港和越南

#### 🇻🇳 越南 — 推荐作为**落地市场本地合规主体**

**适合场景**：
- 越南本地进出口交易（必须有报关单）
- 需要开具越南红色发票的交易
- 本地客户的法币收付

**核心优势**：
- **唯一能取得报关单**：越南法律要求进口必须由本地企业报关
- **人员成本低**：4人团队年成本仅3-4万美元
- **外资设立成熟**：流程标准化，4-8周可完成

**关键注意**：
- **有外汇管制**：每笔跨境收付需银行审核合同+发票+报关单
- 资金汇出需提供完整三单匹配（合同+发票+银行水单）
- 外资企业必须年度审计
- 至少1名越南籍员工

### 3.4 推荐组合方案

根据不同业务需求，推荐以下法域组合：

| 方案 | 组合 | 适用场景 | 优先级 |
|------|------|---------|-------|
| **方案一（轻量起步）** | 香港 | 仅需资金归集中心，不涉及本地报关 | ⭐⭐⭐⭐⭐ 首选 |
| **方案二（合规+归集）** | 香港 + 新加坡 | 香港做资金归集，新加坡做合规背书 | ⭐⭐⭐⭐ |
| **方案三（含越南）** | 香港 + 越南 | 有越南本地交易需求（需报关单） | ⭐⭐⭐⭐ |
| **方案四（全覆盖）** | 香港 + 新加坡 + 越南 | 全亚太覆盖 | ⭐⭐⭐ 中后期 |

**建议执行顺序**：先设香港公司（1-2个月见效）→ 按需增设新加坡或越南

## 4. 运营架构设计

### 4.1 合同体系

| 合同类型 | 签约方 | 核心条款 |
|---------|--------|---------|
| 代理采购合同 | 代收付主体 ↔ 买方 | 采购标的、金额、付款方式、代理费 |
| 代理销售合同 | 代收付主体 ↔ 卖方 | 供货标的、金额、结算方式、佣金 |
| 服务协议 | 代收付主体 ↔ 持牌主体 | 资金归集、结算周期、服务费、合规义务 |
| 居间服务合同 | 代收付主体 ↔ 买方/卖方 | 居间服务范围、费用、免责条款 |

### 4.2 资金流转路径

```
买方银行账户
    │ ① 买方按合同付款
    ▼
代收付主体银行账户（当地银行）
    ├── ② 扣除代理费/服务费（代收付主体收入）
    ├── ③ 按合同付款给卖方 → 卖方银行账户
    └── ④ 剩余资金按服务协议归集 → 持牌主体/SP账户
```

### 4.3 合规文件管理

| 文件类型 | 用途 | 管理要求 |
|---------|------|---------|
| 进出口报关单 | 证明贸易真实性 | 每笔交易留档，保存5年以上 |
| 商业发票 | 付款依据 | 与合同金额一致，含税号 |
| 提单/运单 | 货物流转证明 | 与报关单交叉验证 |
| 合同原件 | 法律依据 | 双语版本，经公证（如需） |
| 银行水单 | 资金流向证明 | 与合同、发票三单匹配 |
| 税务申报表 | 纳税证明 | 按当地要求按期申报 |

### 4.4 业务比重设计 —— 代收付主体 vs 正常商户

#### 为什么不能全部走代收付主体？

**对 SRO 交代**：如果持牌主体的所有业务都来自几个代收付主体，SRO 会质疑——

> "你的客户全是几个贸易公司，这不正常。你们是不是在用这些公司洗交易？"

**对渠道/SP 交代**：越南银行、SP 也会审查——

> "你这个公司每月几百笔交易，全是代收代付，没有自有业务，是不是通道公司？"

**结论：必须保持健康的业务结构——代收付主体只承接"正常渠道过不了"的部分，大部分业务仍走正常商户路径。**

#### 推荐业务比重

| 业务类型 | 走持牌主体（正常商户） | 走代收付主体 | 说明 |
|---------|---------------------|-------------|------|
| **低风险标准客户** | ✅ 100% | 0% | KYC/KYB 通过、交易背景清晰、SRO 无异议 |
| **中风险客户** | ✅ 70-80% | 20-30% | 主体业务走持牌主体，部分敏感交易走代收付 |
| **高风险地区/复杂背景** | ❌ 0% | ✅ 100% | SRO 不批准的客户/交易类型 |
| **越南本地（需报关单）** | ❌ 0% | ✅ 100% | 必须有本地主体才能取得合规文件 |

#### 总量比重控制建议

```
理想状态：
┌─────────────────────────────────────────────────┐
│            持牌主体（正常商户路径）                   │
│                   60-80%                         │
│  标准 KYC 客户、低风险交易、SRO 友好型业务           │
└─────────────────────────────────────────────────┘
                        +
┌─────────────────────────────────────────────────┐
│              代收付主体路径                         │
│                   20-40%                         │
│  高风险地区客户、复杂贸易背景、本地报关需求            │
└─────────────────────────────────────────────────┘
```

**关键比例红线**：

| 指标 | 安全区 | 警戒区 | 危险区 |
|------|--------|--------|--------|
| 代收付主体占持牌主体总收入比重 | < 30% | 30-50% | > 50% |
| 单个代收付主体占持牌主体收入 | < 15% | 15-25% | > 25% |
| 代收付主体占渠道/银行交易量 | < 40% | 40-60% | > 60% |

> **原则：代收付主体是"补充渠道"，不是"主渠道"。** 持牌主体必须有足够多的正常商户，代收付主体只是处理边缘场景。

#### 向各方的说辞设计

**对 SRO 审计**：
- "我们的客户结构中，80% 是直接 KYC 通过的标准商户"
- "与 XX 贸易公司（代收付主体）的业务合作，是基于独立商业服务协议的正常 B2B 贸易结算"
- "该贸易公司是我们的企业客户之一，经过标准 KYB 审核"
- 提供：代收付主体的 KYB 文件、商业合同、发票、交易明细

**对越南银行/渠道**：
- "我们是一家国际贸易代理公司，帮助海外买家采购越南商品"
- "我们的客户包括 XX 个进口商，月均 XX 笔交易"
- 提供：进口合同、报关单、发票、客户名单
- **关键：代收付主体在越南银行眼中，应该是一家正常运营的贸易公司，有多个上下游客户，而不是只对接一个持牌主体**

**对新加坡/香港银行**：
- "我们是一家商务咨询和贸易代理公司"
- "客户覆盖 XX 个国家，涉及 XX 个行业"
- 提供：服务协议、代理合同、客户清单（含非持牌主体的其他客户）

### 4.5 代收付主体数量规划

#### 成立几个合适？

**核心原则：分散风险 + 控制成本 + 业务合理性**

| 因素 | 少（1-2个） | 适中（3-4个） | 多（5+个） |
|------|------------|-------------|-----------|
| **管理复杂度** | ✅ 低 | 🟡 可控 | ❌ 高 |
| **设立+运营成本** | ✅ 低 | 🟡 中 | ❌ 高 |
| **单点风险** | ❌ 高（一个被关户全停） | ✅ 分散 | ✅ 充分分散 |
| **银行审查压力** | ❌ 单个主体交易量大，易触警 | ✅ 每个主体交易量适中 | 🟡 太多主体反而引起注意 |
| **SRO视角** | ✅ 关联方少 | 🟡 需解释多个合作方 | ❌ 多到不合理 |
| **渠道视角** | ❌ 集中度高 | ✅ 正常 | 🟡 需解释 |

#### 推荐方案

**起步阶段（0-6月）：2个主体**

| # | 法域 | 定位 | 月交易量目标 |
|---|------|------|------------|
| 1 | 🇭🇰 香港 | 亚太资金归集 + 通用合同载体 | 50-200万 USD |
| 2 | 🇻🇳 越南（如需） | 越南本地报关 + 法币收付 | 20-100万 USD |

**成长阶段（6-12月）：3-4个主体**

| # | 法域 | 定位 | 月交易量目标 |
|---|------|------|------------|
| 1 | 🇭🇰 香港 A | 亚太区贸易代理（行业A） | 100-300万 USD |
| 2 | 🇭🇰 香港 B | 亚太区贸易代理（行业B） | 100-300万 USD |
| 3 | 🇸🇬 新加坡 | 东南亚枢纽 + 高合规背书 | 50-200万 USD |
| 4 | 🇻🇳 越南 | 本地报关 | 20-100万 USD |

> **为什么香港建议开两个？**
> - 分散银行账户风险（一个被冻结另一个继续运营）
> - 按行业/客户类型分开，每个主体的业务画像更清晰
> - 单个主体月交易量控制在合理范围，不触发银行风控

#### 每个主体的交易量控制

| 法域 | 单个主体月交易量建议上限 | 原因 |
|------|----------------------|------|
| 🇭🇰 香港 | 300-500万 USD | 超过此量级银行会要求加强审查或提升合规等级 |
| 🇸🇬 新加坡 | 200-300万 USD | 新加坡银行对贸易公司资金流更敏感 |
| 🇻🇳 越南 | 100-200万 USD | 越南外汇管制+银行审查，量不宜过大 |

> **如果交易量超过上限** → 要么增设同法域新主体，要么提升现有主体的银行合规等级（提供更完整的审计报告、增加注册资本等）

#### 主体间关系设计

```
                    ┌─────────────────────────┐
                    │    瑞士 SRO 持牌主体       │
                    │  （正常商户 60-80%）        │
                    └────┬───────────┬─────────┘
                         │           │
              独立服务协议A │           │ 独立服务协议B
                         ▼           ▼
                 ┌──────────┐  ┌──────────┐
                 │ 🇭🇰 香港 A │  │ 🇭🇰 香港 B │
                 │ 行业A客户  │  │ 行业B客户  │
                 └──────────┘  └──────────┘
                                    │
                         独立服务协议C │
                                    ▼
                            ┌──────────┐
                            │ 🇻🇳 越南   │
                            │ 本地报关   │
                            └──────────┘

关键：
- 各代收付主体之间无股权关联
- 各代收付主体与持牌主体之间各自签独立协议
- 代收付主体之间可以有业务往来（如香港A委托越南报关）
  但必须有独立合同支撑
```

```

#### 设立流程

| **注册** | 第1-2周 | 公司名核准、提交注册、获取 CI（公司注册证）+ BR（商业登记证） |

1. **年审**：每年提交周年申报表（AR1）+ 商业登记费

- 与瑞士持牌主体之间签订独立的商业服务协议
- 香港公司独立开具发票、独立做账
- 瑞士SRO审计时，香港公司不在审计范围内

- 新加坡公司以贸易公司身份运营，不触发 MAS 监管
- 与瑞士持牌主体之间为独立商业合作关系

## 6. 风险分析

### 6.0 瑞士 SRO 相关风险（最重要）

| 风险 | 等级 | 描述 | 应对措施 |
|------|------|------|---------|
| **SRO认定关联** | 🔴高 | SRO审计时发现代收付主体与持牌主体存在实质关联（相同股东、相同管理层、资金频繁互转），可能要求将代收付主体纳入监管范围 | ①股权完全独立 ②管理层不重叠 ③资金流转有独立商业合同支撑 ④代收付主体有独立决策记录 |
| **资金链路可追溯** | 🔴高 | SRO要求持牌主体说明所有资金来源；如果代收付主体是主要资金来源方，SRO可能要求穿透审查 | ①代收付主体归集后的资金以"商业服务款"名义支付给持牌主体 ②附完整合同和发票 ③资金来源多元化，不依赖单一代收付主体 |
| **"规避监管"定性** | 🔴高 | 如果SRO认定设立代收付主体的目的是为了规避监管要求，可能导致牌照处罚 | ①代收付主体有独立的商业目的（贸易代理、市场拓展） ②不仅仅是资金通道 ③有实际贸易背景和客户关系 |
| **SRO政策收紧** | 🟡中 | SRO可能修改规则，扩大关联方认定范围或加强资金来源审查 | ①持续跟踪SRO政策动态 ②预留调整方案 ③确保代收付主体能独立存续 |

### 6.1 法律风险

| 风险 | 等级 | 描述 | 应对措施 |
|------|------|------|---------|
| **主体穿透** | 🔴高 | 监管机构穿透代收付主体，认定实际控制人为持牌主体，将监管延伸 | ①股权上独立（不由持牌主体直接持股）②保持经营独立性 ③合同设计为平等商业合作 |
| **合同效力** | 🟡中 | 合同被认定为"虚假贸易"，导致无效、资金冻结 | ①确保每笔有真实贸易背景 ②保留货物/服务交付证据 ③价格不偏离市场 |
| **无照经营** | 🟡中 | 某些国家可能认定代收付业务属于"支付服务"需牌照 | ①经营范围定义为"贸易代理" ②取得当地法律意见书 ③持续关注政策变化 |
| **关联交易** | 🟡中 | 与持牌主体资金流转被认定关联交易，触发税务稽查 | ①独立定价，留存转让定价文档 ②服务费率合理 ③避免无商业理由的频繁流转 |

### 6.2 运营风险

| 风险 | 等级 | 描述 | 应对措施 |
|------|------|------|---------|
| **银行账户** | 🔴高 | 资金流动异常（频繁大额进出）导致冻结/关户 | ①提前与银行沟通业务模式 ②交易金额与合同匹配 ③多银行开户分散风险 ④定期提供业务说明 |
| **团队管理** | 🟡中 | 远程管理当地团队，操作不合规、信息不对称 | ①标准化SOP ②财务系统实时可见 ③定期审计 ④核心签署权限总部控制 |
| **资金滞留** | 🟡中 | 外汇管制导致资金无法及时汇出 | ①了解当地外汇管制政策 ②控制账户余额及时归集 ③预留应急通道 |

### 6.3 税务风险

| 风险 | 等级 | 描述 | 应对措施 |
|------|------|------|---------|
| **常设机构认定** | 🟡中 | 税务机关认定持牌主体通过代收付主体构成"常设机构"(PE) | ①代收付主体独立决策 ②避免"分支机构"措辞 ③取得税务顾问意见 |
| **转让定价** | 🟡中 | 服务费过高/过低被税务机关调整 | ①参照市场可比价格 ②准备转让定价文档 ③费率在1%-5%合理区间 |
| **双重征税** | 🟢低 | 同一笔收入在两个法域被重复征税 | ①利用双边税收协定(DTA) ②合理规划收入确认时点 |

### 6.4 合规风险

| 风险 | 等级 | 描述 | 应对措施 |
|------|------|------|---------|
| **反洗钱** | 🔴高 | 代收付模式天然涉及资金中转，易被怀疑为洗钱通道 | ①严格KYC了解买卖双方真实身份 ②每笔有真实贸易背景 ③大额主动报告 ④建立内部AML制度 |
| **监管政策变化** | 🟡中 | 当地可能出台新法规要求代收付业务持牌 | ①持续跟踪法规变化 ②预留牌照申请窗口 ③与当地律所长期合作 |

### 6.5 风险总览

| 等级 | 风险项 | 核心对策 |
|------|--------|---------|
| 🔴 高 | **SRO认定关联** | 股权独立 + 管理层不重叠 + 独立决策记录 |
| 🔴 高 | **SRO资金链路追溯** | 商业合同支撑 + 资金来源多元化 |
| 🔴 高 | **"规避监管"定性** | 代收付主体有独立商业目的 + 真实贸易背景 |
| 🔴 高 | 主体穿透 | 股权隔离 + 经营独立 |
| 🔴 高 | 银行账户 | 多银行 + 合规沟通 |
| 🔴 高 | 反洗钱 | 严格KYC + 交易留痕 |
| 🟡 中 | SRO政策收紧 / 合同效力 / 无照经营 / 关联交易 | 法律意见书 + 持续跟踪 |
| 🟡 中 | 团队管理 / 资金滞留 | SOP + 实时监控 |
| 🟡 中 | 常设机构 / 转让定价 / 政策变化 | 税务顾问 + 持续跟踪 |
| 🟢 低 | 双重征税 | DTA协定 |

## 8. 决策建议

| 条件 | 是否增设新加坡 |

### 按需 —— 越南

| 条件 | 是否增设越南 |

| 有越南本地进出口交易（需报关单） | ✅ 必须 |

| 无越南本地交易 | ❌ 不需要 |

### 长期（12月+）

- 形成 **香港（归集）+ 新加坡（合规背书）+ 越南（本地落地）** 的三角架构
- 各代收付主体独立运营，与瑞士持牌主体仅有商业合作关系
- 交易数据沉淀 → 反哺瑞士持牌主体的客户筛选和风控

                   （完全无股权关联）
                          ×  ← 不持股
                          
自然人A ──┬── 60% → 🇭🇰 香港代收付主体
          └── 60% → 🇸🇬 新加坡代收付主体

自然人B ──┬── 40% → 🇭🇰 香港代收付主体
          └── 40% → 🇸🇬 新加坡代收付主体

自然人C ── 100% → 🇻🇳 越南代收付主体（如需）
```

**优点**：SRO审计时无法通过股权找到关联；各主体独立运营
**风险**：自然人个人承担责任；多法域管理复杂度

### 方案B —— 离岸控股统一管理

```
自然人A/B ── 持有 → 离岸控股(BVI)
                        │
              ┌─────────┼─────────┐
              ▼         ▼         ▼
          🇭🇰 香港   🇸🇬 新加坡   🇻🇳 越南
```

**优点**：便于多国复制、统一管理、税务规划空间
**风险**：穿透较容易；SRO可能追溯到离岸控股层面

### 核心原则

> **代收付主体在股权上不应由瑞士 SRO 持牌主体直接或间接持有。**
> 
> 持牌主体与代收付主体之间的唯一关系 = 独立商业服务协议（平等合作方），而非母子公司或关联方。
> 
> 建议在设立前咨询瑞士律师，确认结构设计满足 SRO 审计要求。
