4发卡全流程

1 商户入网 + 产品开通
推送商户入网→ 转发 SP→ KYC/KYB 审核→ KYB_APPROVED→ 申请 VCC 产品→ PRODUCT_ACTIVE
• 商户由租户创建，通过 EX 推送至 SP
• SP 执行 KYC/KYB 审核（可能触发 RFI 补充材料）
• 审核通过后申请 VCC 产品开通
2 法币充值（VA 路径）
申请 VA→ VA 审核→ VA_ACTIVE→ 线下银行转账→ 入账风控→ FIAT_DEPOSIT_SUCCESS
ℹ️ VA 创建后获取 {accountNumber, bankInfo}，租户线下向该 VA 转账即可
3 数币充值（充币地址路径）
获取充币地址→ 返回 {address, chain}→ 链上转入 USDT/USDC→ 链上确认 + AML 风控→ CRYPTO_DEPOSIT_SUCCESS
4 开卡 — 共享卡（首次）
创建共享账户→ 共享账户充值→ SHARED_ACCOUNT_FUNDED→ 发共享卡→ CARD_ISSUED
首次需创建共享账户并充值；后续复用已有共享账户直接发卡
5 开卡 — 共享卡（复用）
查询已有共享账户→ 发卡（选已有账户）→ CARD_ISSUED
6 开卡 — 充值卡
发充值卡 + 充值→ 风控审核→ CARD_ISSUED_AND_FUNDED
一步完成发卡 + 充值，无需额外操作
7 卡消费与交易同步
持卡人消费→ 卡组织授权→ SP 校验余额 + 风控→ 清算扣款→ CARD_TRANSACTION
消费完成后 EX 自动推送 Webhook，租户同步交易流水

5RFI（补充材料）处理

SP 风控需补材料→ RFI_REQUEST→ 上传文件→ 提交 RFI→ 转发 SP→ RFI_RESOLVED / REJECTED
场景可能要求的文件
商户入网 KYB营业执照、董事身份证明、业务说明
VA 申请资金来源证明、业务合同
大额法币充值银行流水、贸易单据
数币充值链上资金来源证明
开卡审核持卡人身份证明、消费用途说明
