4发卡全流程

1 商户注册 + 开通产品（含 KYC/KYB）
第一步：商户注册
注册商户（基本信息）→ EX 创建商户记录→ 返回商户标识
仅在 EX 侧注册，不涉及 SP、不需要 KYC/KYB
第二步：开通产品 + KYC/KYB（首次需客户入网）
申请 VCC 产品 + KYB 资料→ 转发 SP→ KYC/KYB 审核→ KYB 通过 + VCC 已开通
• 首次入网需提交 KYB 资料，SP 审核通过后产品开通
• 非首次入网（已有 KYB）只需申请开通产品，无需重复 KYB
• 可能触发 RFI 补充材料
2 法币充值（VA 路径）
申请 VA→ VA 审核→ VA 已激活→ 线下银行转账→ 入账风控→ 法币充值成功
ℹ️ VA 创建后获取 {accountNumber, bankInfo}，租户线下向该 VA 转账即可
3 数币充值（充币地址路径）
获取充币地址→ 返回充币地址→ 链上转入 USDT/USDC→ 链上确认 + AML 风控→ 数币充值成功
4 开卡 — 共享卡（首次）
创建共享账户→ 共享账户充值→ 共享账户充值成功→ 发共享卡→ 发卡成功
首次需创建共享账户并充值；后续复用已有共享账户直接发卡
5 开卡 — 共享卡（复用）
查询已有共享账户→ 发卡（选已有账户）→ 发卡成功
6 开卡 — 充值卡
发充值卡 + 充值→ 风控审核→ 发卡+充值成功
一步完成发卡 + 充值，无需额外操作
7 卡消费与交易同步
持卡人消费→ 卡组织授权→ SP 校验余额 + 风控→ 清算扣款→ 卡消费通知
消费完成后 EX 自动推送 Webhook，租户同步交易流水

5RFI（补充材料）处理

SP 风控需补材料→ 补充材料请求→ 上传文件→ 提交补充材料→ 转发 SP→ 审核通过 / 拒绝
场景可能要求的文件
商户入网 KYB营业执照、董事身份证明、业务说明
VA 申请资金来源证明、业务合同
大额法币充值银行流水、贸易单据
数币充值链上资金来源证明
开卡审核持卡人身份证明、消费用途说明
