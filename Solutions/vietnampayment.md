#### 2\. Setup Fee 讨论 ⭐ 重要

    - 如果 setup fee 是 5K 美金，对他们来说太高

- **9pay 的策略**：

    - 一家银行对接 1\-2 家银行

    - 每笔交易展示**最优价格和最优费率**

    - 竞争激烈，必须做"best price"

    - **结论：可以协商 setup fee 或 yearly fee，但必须达到约定量**

- **9pay vs Nanpei 对比**：

    - **Nanpei**：不收 setup fee、monthly fee、other fees（很竞争）

    - **9pay**：会收 setup fee 和 minimum fee 给其他 TSP

    - **客户反馈**：很多越南客户愿意为 setup fee 付费（接受度在提高）

- **建商付 setup fee 的策略**：

    - 起步价 0%（鼓励测试）

    - 月交易量达到 5M USDT 后减到 0\.7%

    - 达到 25M USDT 后降至 0\.6%

    - 最低 0\.6%（公司与个人都适用）

    - 需求条件：要先提供交易量证明
