## 2.2 实战案例 / Proof Points

> **三大角色分工** （每个案例均由三者协同完成）：
>
> * **iPayLinks —— 全球持牌****法币** **清结算网络** ：全球收付、换汇、跨境清算与人民币在岸结汇。
> * **BlancBlock —— 稳定币清结算与流动性网络** ：法币 ⇄ 稳定币（USDC/USDT）流动性、做市与链上实时清算。
> * **EurewaX —— AI 驱动的全栈科技底座** ：清算引擎、实时 FX、统一账簿、智能风控，统一编排资金路由（记账、对账、查汇 / 锁汇、风控）。
> * 每个案例按" **付款方 → 资金路由（** **法币** ** iPayLinks / 数币 BlancBlock）→ 收款方与到账币种** "展开。

### 案例一 · 跨境 B2B：越南买家付款、中国卖家收款

* **通俗讲** ：越南客户用本地的钱付款，中国卖家国内账户直接收到人民币。
* **场景** ：越南买家以本地货币或稳定币付款，中国卖家在境内对公账户收到人民币，换汇、合规与清算由平台一体完成。
* **痛点** ：传统电汇到账周期 3–5 天、手续费高、美元额度受限；汇率波动直接侵蚀薄利贸易的毛利。
* **方案 · 资金流** ：
* 越南买家本地付款：**法币****VND** 或  **数币 USDT** 。
* 法币侧 →  **iPayLinks** ：收 VND，换汇 **VND → ****USD**** → ** **CNY** ，完成跨境清算。
* 数币侧 →  **BlancBlock** ：收 USDT，链上清算（按需 **USDT → ****USD**** → ** **CNY** ）。
* 中国卖家收款  **CNY** （在岸结汇至对公账户，支持贸易申报与退税）。
* **EurewaX** 全程负责记账、账务、查汇 / 锁汇、合规风控与对账。
* **客户价值** ：当日 / 次日到账、综合成本下降、订单级自动对账、全流程可追溯。

### 案例二 · 多种方式支付、多种方式结算：消费者怎么付都行，商家想收什么币都行

* **通俗讲** ：消费者怎么付都行，商家想收什么币都行，中间的换汇与清算由我们一站搞定。
* **场景** ：消费者用本地方式与货币付款（卡 / 扫码 / 本地钱包 / 数币），商家按需选择法币或数币收款（如 USD / CNH / CNY / USDC / USDT）。
* **痛点** ：消费者支付方式分散、跨币种结算复杂；商家希望用合适的币种收款，但换汇与清算又慢又贵。
* **方案 · 资金流向** ：
  1. 消费者用 **任意方式** 付款——卡 / 本地扫码 / 本地钱包 / **数币**。
  2. 法币侧 → **iPayLinks** ：收款并完成换汇 + 跨境清算。
  3. 数币侧 → **BlancBlock** ：收数币并完成链上清算。
  4. 商家以 **自选币种** 到账——法币（USD / CNH / CNY）或数币（USDC / USDT）。
  5. **EurewaX** 全程编排换汇、风控与对账。
* **客户价值** ：消费者支付方式全覆盖、商家结算币种自由选择（法币或数币），换汇清算又快又省。

### 案例三 · OTA 发卡付款：向酒店 / 航司批量结算

* **通俗讲** ：给旅游平台发“虚拟卡”，按订单刷卡付给酒店、航司，账自动对平。
* **场景** ：OTA 平台需向大量酒店、航司付款，平台按订单 / 额度发行虚拟卡完成支付，并实现自动对账。
* **痛点** ：上游供应商分散，逐笔付款与对账成本高、资金管控与安全难度大。
* **方案 · 资金流** ：
* OTA 平台资金充值进清结算中心：**法币** **经 iPayLinks** 、 **数币经 BlancBlock** 。
* **EurewaX** 按订单 / 额度 **发行虚拟卡** （法币卡 / 数币卡），并完成记账与额度管控。
* 以卡向酒店 / 航司付款；需换汇时 **iPayLinks** 处理法币换汇、**BlancBlock** 处理数币清算。
* **EurewaX** 全程自动对账、风控与额度管理。
* **客户价值** ：按需发卡、额度可控、对账自动化、资金安全可追溯。

### 案例四 · 锁汇产品：买家付美金，卖家用人民币与工厂结算

* **通俗讲** ：买家付的是美金，但卖家要用人民币付给工厂和物流商；卖家在签约当天锁定美元换人民币的汇率，几个月后人民币成本不变。
* **场景** ：美国买家以 **USD** 报价采购，中国卖家则需以 **CNY** 与 **工厂 / 物流服务商** 结算；签约至收款周期内的汇率波动直接侵蚀卖家毛利。
* **痛点** ：买家付款币种（USD）与卖家成本币种（CNY）不一致，叠加 30–60 天账期，汇率波动给卖家带来不确定的毛利风险。
* **方案 · 资金流** ：
  1. 买家以 **USD** 报价下单；签约当日 **由卖家通过 EurewaX 锁汇** ，锁定 **USD → CNY** 汇率（30–60 天），以保住人民币成本。
  2. 收款：买家支付 USD（法币经 **iPayLinks** 收；若以稳定币支付则经 **BlancBlock** 收并清算）。
  3. 到期按锁定汇率换汇：**iPayLinks** 执行 **USD → CNY** 。
  4. 卖家到账 **CNY** ，并以人民币付给 **工厂 / 物流服务商** ；**EurewaX** 全程记账、查汇、风控。
* **客户价值** ：锁定卖家人民币成本与毛利、规避汇率波动、对工厂 / 物流的付款金额可预期。

## 2.3 为何我们能做跨境资金清结算 / Why We Can Be the Settlement Hub
