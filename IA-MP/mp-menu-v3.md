# 菜单结构 v3（MP + TP）

---

## 一、MP 商户端菜单

> 分组逻辑：按业务线分组，配置与订单放同一业务下，链路完整。
> 商户自己用，SP 不可见，只看自己的数据。

### 菜单总览

| 分组               | 子菜单                                 | 说明                                    |
| ------------------ | -------------------------------------- | --------------------------------------- |
| Assets（资产管理)  | Fiat Accounts / Crypto Wallet          | 看余额、充提（同名充值/提现/充币/提币） |
| Directory（通讯录) | 收款工具 / 汇款人（本期没有） / 收款人 | 所有工具和对手方统一管理                |
| Payment（收付兑）  | Payins / Payouts / Exchange            | 收款订单 + 付款订单 + 兑换订单          |
| Checkout （收单）  | Online Payment / Invoice               | 商户收单（C端消费者付款）               |
| Cards（卡）        | My CardCards  / Card Transactions     | 发卡 + 卡消费记录                       |

### 详细结构

#### 1. 资产管理（Assets）

| 菜单项        | 功能                                   |
| ------------- | -------------------------------------- |
| Fiat Accounts | 法币账户余额总览，按币种聚合（SP透明） |
| Crypto Wallet | 数币钱包余额总览                       |

> 充值/提现/充币/提币 = 账户产品（配套能力），在账户详情页内发起，不单独做菜单入口。

#### 2. 通讯录（Directory）

| 菜单项                       | 功能                                              |
| ---------------------------- | ------------------------------------------------- |
| 收款工具（Collection Tools） | 收款银行账户（VA）+ 收款地址（Crypto）            |
| ~~汇款人（Remitter）~~      | ~~汇款人维护（暂未开发）~~                       |
| 收款人（Beneficiary）        | 同名账户（提现用）/ 第三方账户（付款用）+收款地址 |

> 充值 VA 和收款 VA 是同一个 VA，系统靠汇款人匹配区分充值/收款。
> Beneficiary 区分同名（Own Account）和第三方（Third Party）。

#### 3. 收付兑业务

| 菜单项                  | 功能                                                      |
| ----------------------- | --------------------------------------------------------- |
| 收款订单（Collections） | 所有入金记录（VA收款 + 链上收款 + 同名充值 + 同名充币）   |
| 付款订单（Payouts）     | 所有出金记录（法币付款 + 链上付币 + 同名提现 + 同名提币） |
| 汇兑订单（Exchange）    | FX（法币换汇）+ OnRamp（买入数币）+ OffRamp（卖出数币）   |

> 买入/卖出数币本质是"兑换"（资产形态转换），没有外部对手方，归兑换业务。

#### 4. 收单业务（Checkout）

| 菜单项         | 功能                                    |
| -------------- | --------------------------------------- |
| Online Payment | 在线收单订单（C端消费者通过收银台付款） |
| ~~Invoice~~  | ~~账单/发票收款~~                     |

> 收单 vs 收款：收单有独立结算周期，结算后才进账户余额；收款（Payins）实时入账。

#### 5. 卡业务（Cards）

| 菜单项            | 功能                           |
| ----------------- | ------------------------------ |
| My Cards          | 卡管理：开卡、充值、冻结、销卡 |
| Card Transactions | 卡订单记录：消费、退款、ATM    |

## 二、TP 支付机构运营后台菜单

> 分组逻辑：按运营职能分组，管所有商户的数据，SP 可见。
> 支付机构运营人员用，审核/配置/监控。

### 菜单总览

| 分组                         | 子菜单                                        | 说明                                        |
| ---------------------------- | --------------------------------------------- | ------------------------------------------- |
| 工作台（Dashboard）          | Dashboard                                     | 待办、今日数据、异常告警                    |
| 商户管理（Merchants）        | Merchant List / Products / Configuration      | 管"人"：入网、KYB、产品审批                 |
| 账户 & 资产（Accounts）      | Balances / Deposit & Withdraw                 | 管"钱"：所有商户余额（含 SP 明细）          |
| 通讯录（Directory）          | Collection Tools / Remitter / Beneficiary     | 管"工具和对手方"：VA、Remitter、Beneficiary |
| 收付兑业务（Payments）       | Collections / Payouts / Exchange              | 管"单"：所有商户的收付兑流水                |
| 收单业务（Checkout）         | Checkout Orders / Settlements                 | 管"收单"：商户收单交易                      |
| 卡业务（Cards）              | Card Issuing / Card Transactions              | 管"卡"：开卡审批、卡消费                    |
| 费率 & 结算（Pricing）       | Product Pricing / Merchant Pricing / Rules    | 管"价格"                                    |
| 财务（Finance）              | Receivables / Payables / Statements / Recon   | 管"账"                                      |
| 渠道管理（Channels）         | SP Channels / Routing Rules                   | 管"渠道"                                    |
| 系统设置（Settings）         | Roles / Users / Tickets                       | 管"系统"                                    |

### 详细结构

#### 1. 工作台

Dashboard：待办事项、今日交易数据、异常告警。

#### 2. 商户管理（Merchants）

| 菜单项                        | 功能                                 |
| ----------------------------- | ------------------------------------ |
| 商户列表（Merchant List）     | 所有商户基本信息、KYB 状态、联系方式 |
| 产品开通（Products）          | 商户已开通产品及状态、审批           |
| 商户配置（Configuration）     | 限额、风控参数、渠道分配             |

#### 3. 账户 & 资产（Accounts）

| 菜单项                          | 功能                                                                         |
| ------------------------------- | ---------------------------------------------------------------------------- |
| 余额总览（Balances）            | 所有商户账户/钱包余额（可见 SP 明细），冻结/解冻操作                         |
| 充提记录（Deposit & Withdraw）  | 商户充值/提现流水（充值 :Fiat/CryptoDEPOSIT，提现 = :Fiat/CryptoWITHDRAWAL） |

> 充提记录是从 collections/Payouts 按类型过滤的视图，底层同表。

#### 4. 通讯录（Directory）

| 菜单项                       | 功能                                                            |
| ---------------------------- | --------------------------------------------------------------- |
| 收款工具（Collection Tools） | 商户 VA + 加密收款地址管理（分配/停用/状态）                    |
| ~~汇款人（Remitter）~~      | ~~汇款人审核/管理（暂未开发）~~                                |
| 收款人（Beneficiary）        | 同名账户（Own Account）/第三方账户（Third Party）管理+ 提币地址 |

#### 5. 收付兑业务（Payments）

| 菜单项                  | 功能                                    |
| ----------------------- | --------------------------------------- |
| 收款订单（Collections） | 所有商户收款流水：VA收款 + 链上收款     |
| 付款订单（Payouts）     | 所有商户付款流水：法币付款 + 链上付币   |
| 汇兑订单（Exchange）    | 所有商户兑换流水：FX + OnRamp + OffRamp |

#### 6. 收单业务（Checkout）

| 菜单项          | 功能                       |
| --------------- | -------------------------- |
| Checkout Orders | 商户收单交易记录、退款审批 |
| Settlements     | 收单结算记录               |

#### 7. 卡业务（Cards）

| 菜单项                         | 功能                        |
| ------------------------------ | --------------------------- |
| 发卡管理（Card Issuing）       | 开卡审批、卡状态、冻结/销卡 |
| 卡交易（Card Transactions）    | 卡消费/退款/ATM 记录        |

#### 8. 费率 & 结算（Pricing）

| 菜单项                          | 功能               |
| ------------------------------- | ------------------ |
| 产品费率（Product Pricing）     | 各产品标准费率配置 |
| 商户费率（Merchant Pricing）    | 逐商户定制费率     |
| 结算规则（Settlement Rules）    | 结算周期、方式配置 |
| 分佣规则（Commission Rules）    | 代理/渠道分润配置  |

#### 9. 财务（Finance）

| 菜单项                       | 功能                          |
| ---------------------------- | ----------------------------- |
| 应收（Receivables）          | 向商户收取的手续费/佣金       |
| 应付（Payables）             | 付给商户的返点/付给代理的分润 |
| 结算单（Statements）         | 商户结算账单                  |
| 对账（Reconciliation）       | SP 通道对账                   |

#### 10. 渠道管理（Channels）

| 菜单项                     | 功能                       |
| -------------------------- | -------------------------- |
| SP 通道（SP Channels）     | 服务商通道状态/配额/健康度 |
| 路由规则（Routing Rules）  | 交易路由配置               |

#### 11. 系统设置（Settings）

| 菜单项               | 功能             |
| -------------------- | ---------------- |
| 角色管理（Roles）    | 内部员工角色权限 |
| 用户管理（Users）    | 内部员工账号     |
| 工单（Tickets）      | 商户工单处理     |

## 三、产品定义说明

### 账户产品 vs 业务产品

| 类型                 | 产品                         | 性质                                 |
| -------------------- | ---------------------------- | ------------------------------------ |
| 账户产品（配套能力） | 充值、提现、兑换、账户间划转 | 自己的钱在自己账户里动，无外部对手方 |
| 业务产品（独立开通） | 收款、付款、收单、发卡       | 跟外部对手方产生资金往来             |

### 产品开通组合

| 开通产品            | 必然附带      | 不一定开         |
| ------------------- | ------------- | ---------------- |
| 收单（Checkout）    | + 提现 + 账户 | 付款、收款、发卡 |
| 收款（Collections） | + 提现 + 账户 | 付款、收单、发卡 |
| 付款（Payouts）     | + 充值 + 账户 | 收款、收单、发卡 |
| 发卡（Cards）       | + 充值 + 账户 | 收款、付款、收单 |
| 兑换（Exchange）    | + 账户        | 可单独开         |

### VA 和 Beneficiary 归属

- **VA** = 收款工具，放通讯录。同一个 VA 既接收同名充值也接收第三方收款，系统靠汇款人匹配区分。
- **Beneficiary** = 收款人，放通讯录。区分同名账户（提现目标）和第三方账户（付款目标）。
- 充值/提现不单独做工具管理页面，统一在通讯录管理。

### 买入/卖出数币归属

| 操作                | 本质           | 有对手方？ | 归属     |
| ------------------- | -------------- | ---------- | -------- |
| 买入数币（OnRamp）  | 法币 → 数币   | ❌         | 兑换业务 |
| 卖出数币（OffRamp） | 数币 → 法币   | ❌         | 兑换业务 |
| 法币换汇（FX）      | 法币A → 法币B | ❌         | 兑换业务 |

三者都是"自己的钱在不同形态间转换"，没有收/付对手方，归收付兑业务下的兑换订单。
